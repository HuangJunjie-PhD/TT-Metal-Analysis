## Purpose and Scope

This document covers the implementation of masked language models and text classification models within the tt-forge-models repository. These models represent a distinct category of NLP models that differ from the autoregressive causal language models covered in [Large Language Models](https://deepwiki.com/tenstorrent/tt-forge-models/4.1-large-language-models). The scope includes BERT-style bidirectional models for masked language modeling tasks, sequence classification models, and question answering models that leverage the same underlying architectures.

The models documented here primarily use bidirectional attention mechanisms and are typically fine-tuned for specific downstream tasks, in contrast to the generative causal models used primarily for text generation.

## Tokenization and Input Processing

All models in this category use HuggingFace `AutoTokenizer` for consistent text preprocessing, but with task-specific patterns:

### Masked Language Modeling

*   Uses `[MASK]` tokens for prediction targets
*   Simple text input with mask token placement
*   Returns standard `input_ids` and `attention_mask` tensors

### Text Classification

*   Processes single text sequences
*   Applies padding and truncation to fixed lengths
*   Supports batch size expansion through `repeat_interleave`

### Question Answering

*   Combines question and context using `encode_plus`
*   Uses special token separation for question-context pairs
*   Requires padding side configuration (`padding_side: "left"`)

The tokenizer loading pattern is consistent across implementations, with optional `dtype_override` support for model precision control [albert/masked_lm/pytorch/loader.py 91-111](https://github.com/tenstorrent/tt-forge-models/blob/0a7d2042/albert/masked_lm/pytorch/loader.py#L91-L111)
