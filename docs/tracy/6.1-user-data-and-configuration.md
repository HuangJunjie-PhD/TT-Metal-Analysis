This document covers Tracy's user data and configuration system, which manages persistent user settings, annotations, and customizations for profiling sessions. The system handles storing and retrieving user-specific data including view states, annotations, source path substitutions, and program descriptions.

For information about Tracy file formats and trace data storage, see [File Formats](https://deepwiki.com/tenstorrent/tracy/5.1-file-formats). For command-line utilities that work with user data, see [Command-Line Utilities](https://deepwiki.com/tenstorrent/tracy/6.2-command-line-utilities).

## Overview

Tracy's user data system provides persistent storage for user customizations and configurations across profiling sessions. The system is built around the `UserData` class which manages various types of user-specific data through a file-based storage mechanism.

## Configuration Data Types

### View State Configuration

The `ViewData` structure contains timeline and display configuration settings that persist across sessions.

| Field | Type | Purpose |
| --- | --- | --- |
| `zvStart` | `int64_t` | Timeline zoom start position |
| `zvEnd` | `int64_t` | Timeline zoom end position |
| `frameScale` | `int32_t` | Frame visualization scale |
| `frameStart` | `int32_t` | Frame start offset |
| `drawGpuZones` | `uint8_t` | GPU zone visibility toggle |
| `drawZones` | `uint8_t` | CPU zone visibility toggle |
| `drawLocks` | `uint8_t` | Lock visualization toggle |
| `drawPlots` | `uint8_t` | Plot visibility toggle |
| `onlyContendedLocks` | `uint8_t` | Contended lock filter |
| `drawEmptyLabels` | `uint8_t` | Empty label visibility |
| `drawFrameTargets` | `uint8_t` | Frame target indicators |
| `drawContextSwitches` | `uint8_t` | Context switch visualization |
| `darkenContextSwitches` | `uint8_t` | Context switch appearance |
| `drawCpuData` | `uint8_t` | CPU data visibility |
| `drawCpuUsageGraph` | `uint8_t` | CPU usage graph |
| `drawSamples` | `uint8_t` | Sample visualization |
| `dynamicColors` | `uint8_t` | Dynamic color scheme |
| `forceColors` | `uint8_t` | Force color override |
| `ghostZones` | `uint8_t` | Ghost zone display |
| `frameTarget` | `uint32_t` | Target frame rate |
