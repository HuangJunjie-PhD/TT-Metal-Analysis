This document covers the implementation of Large Language Models (LLMs) in the tt-forge-models repository, specifically focusing on causal language models for text generation and completion tasks. This includes models from the Mistral, Llama, and Falcon families that use the standard `ForgeModel` interface for autoregressive text generation.

For masked language models like BERT and RoBERTa, see [Masked Language Models and Text Classification](/tenstorrent/tt-forge-models/4.2-masked-language-models-and-text-classification). For multimodal models that combine language with other modalities, see [Multimodal Models](/tenstorrent/tt-forge-models/5-multimodal-models).

## Model Variants and Configuration

### Mistral Family Configuration

The Mistral implementation provides three model variants with different parameter counts and capabilities:

| Variant | Pretrained Model Name | Description |
| --- | --- | --- |
| `MISTRAL_7B` | `mistralai/Mistral-7B-v0.1` | Base 7B parameter model |
| `MINISTRAL_3B` | `ministral/Ministral-3b-instruct` | Smaller 3B instruction-tuned variant |
| `MINISTRAL_8B` | `mistralai/Ministral-8B-Instruct-2410` | Larger 8B instruction-tuned variant |

**Sources:** [mistral/pytorch/loader.py34-45](https://github.com/tenstorrent/tt-forge-models/blob/0a7d2042/mistral/pytorch/loader.py#L34-L45)

### Llama Family Configuration

The Llama implementation includes both causal language modeling and sequence classification variants:

| Model Path | Variant | Pretrained Model Name | Task |
| --- | --- | --- | --- |
| `llama/causal_lm/` | `LLAMA_3B` | `meta-llama/Llama-3.2-3B` | Causal LM |
| `llama/` | Base | `meta-llama/Llama-3.2-1B-Instruct` | Sequence Classification |

**Sources:** [llama/causal\_lm/pytorch/loader.py33-37](https://github.com/tenstorrent/tt-forge-models/blob/0a7d2042/llama/causal_lm/pytorch/loader.py#L33-L37) [llama/pytorch/loader.py51](https://github.com/tenstorrent/tt-forge-models/blob/0a7d2042/llama/pytorch/loader.py#L51-L51)

### Falcon Configuration

The Falcon model uses a single base configuration:

* **Model Name:** `tiiuae/Falcon3-1B-Base`
* **Max Length:** 512 tokens
* **Default Input:** "Write a function to calculate the factorial of a number"

**Sources:** [falcon/pytorch/loader.py54-56](https://github.com/tenstorrent/tt-forge-models/blob/0a7d2042/falcon/pytorch/loader.py#L54-L56)
