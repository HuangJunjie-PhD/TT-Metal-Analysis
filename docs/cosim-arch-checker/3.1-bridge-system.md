# Bridge System

Relevant source files
*   [bridge/bridge.cc](https://github.com/tenstorrent/cosim-arch-checker/blob/157c6637/bridge/bridge.cc)
*   [bridge/bridge.h](https://github.com/tenstorrent/cosim-arch-checker/blob/157c6637/bridge/bridge.h)
*   [bridge/bridge_base.h](https://github.com/tenstorrent/cosim-arch-checker/blob/157c6637/bridge/bridge_base.h)

The Bridge System implements the central orchestration layer of the cosim-arch-checker framework. The `cBridge` class serves as the primary coordinator that synchronizes DUT execution with the Whisper ISS reference model and manages architectural state comparison through the Core Arch Checker.

This document covers the implementation and functionality of the bridge layer. For DUT monitoring interface details, see [DUT Monitoring Interface](https://deepwiki.com/tenstorrent/cosim-arch-checker/3.2-dut-monitoring-interface). For Whisper integration details, see [Whisper Integration](https://deepwiki.com/tenstorrent/cosim-arch-checker/3.3-whisper-integration). For Core Arch Checker integration, see [System Architecture](https://deepwiki.com/tenstorrent/cosim-arch-checker/2-system-architecture).

## Architecture Overview

The `cBridge` class inherits from `cBridgeBase` and integrates three major components: the DUT monitoring interface, Whisper ISS client, and Core Arch Checker (CAC).

### Command Line Argument Processing

The bridge system supports multiple configuration options parsed from VPI arguments:

| Argument | Variable | Purpose |
| --- | --- | --- |
| `+bridge_tracer` | `bridge_tracer` | Enable detailed trace output |
| `+testfile=<path>` | `testname_` | ELF binary path for Whisper |
| `+bootcode=<path>` | `bootcode_` | Boot ROM binary path |
| `+whisper_path=<path>` | `whisperPath_` | Whisper ISS executable path |
| `+whisper_json_path=<path>` | `whisperJsonPath_` | Whisper configuration JSON |
| `+arch_checks_exclude=<list>` | `archChecksExclude_` | Resources to exclude from checking |
| `+arch_checks_instr_exclude=<list>` | `archChecksInstrExclude_` | Instructions to exclude from checking |
