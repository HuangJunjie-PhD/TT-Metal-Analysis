This page documents the primary user interface components that form the interactive frontend of Model Explorer's visualization system. These components handle user interaction, information display, and graph manipulation within the Angular-based web application.

For information about the core visualization engine and WebGL rendering, see [Visualization Engine](https://deepwiki.com/tenstorrent/model-explorer/4.2-visualization-engine). For details about the underlying Angular services that power these components, see [Core Services](https://deepwiki.com/tenstorrent/model-explorer/4.1-core-services).

## Component Architecture Overview

The UI components in Model Explorer follow a hierarchical structure where specialized components handle different aspects of user interaction and data presentation. The system is built around the concept of panes, with each pane containing a complete set of UI components for graph visualization and interaction.

## Info Panel System

The `InfoPanel` component serves as the primary information display system, showing detailed information about selected graph elements and providing interactive controls for exploring model structure.

### Info Panel Architecture

**Info Panel Component Structure**

The `InfoPanel` uses a section-based approach where different types of information are organized into collapsible sections. Each section contains `InfoItem` objects that represent individual pieces of data with their associated metadata and display properties.
