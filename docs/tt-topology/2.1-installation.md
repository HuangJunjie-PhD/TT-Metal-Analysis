## Overview

The tt-topology tool is a Python package that requires several system-level dependencies and Tenstorrent-specific libraries. Installation involves setting up Rust toolchain, Python environment, and installing the package with its dependencies.

### Rust Toolchain Requirement

tt-topology requires Rust toolchain for the `pyluwen` dependency compilation. Install Rust before proceeding with Python package installation:

`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | shsource "$HOME/.cargo/env"`
Sources: [README.md 24-28](https://github.com/tenstorrent/tt-topology/blob/f86e46c6/README.md?plain=1#L24-L28)

## Installation Methods

### Standard Installation Flow

Sources: [README.md 22-67](https://github.com/tenstorrent/tt-topology/blob/f86e46c6/README.md?plain=1#L22-L67)

### Basic Installation

1.   **Install Rust toolchain** (required for pyluwen compilation):

`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | shsource "$HOME/.cargo/env"`
2.   **Create Python virtual environment** (optional but recommended):

`python3 -m venv .venvsource .venv/bin/activate`
3.   **Clone repository and install**:

`git clone https://github.com/tenstorrent/tt-topology.gitcd tt-topologypip3 install --upgrade pippip3 install .`

Sources: [README.md 24-48](https://github.com/tenstorrent/tt-topology/blob/f86e46c6/README.md?plain=1#L24-L48)

### Developer Installation

For developers who need to modify the codebase:

1.   **Follow basic installation steps 1-2 above**

2.   **Install development dependencies**:

`pip install --upgrade pippip install pre-commit`
3.   **Install in editable mode**:

`pip install --editable .`
4.   **Install pre-commit hooks** (recommended):

`pre-commit install`

Sources: [README.md 50-67](https://github.com/tenstorrent/tt-topology/blob/f86e46c6/README.md?plain=1#L50-L67)

## Troubleshooting Installation Issues

### Common Issues

1.   **Rust compilation errors**: Ensure Rust toolchain is properly installed and sourced
2.   **Permission errors**: Use virtual environment to avoid system-wide installation conflicts
3.   **Git dependency failures**: Ensure Git is installed and has access to GitHub repositories

For additional troubleshooting information, see [Troubleshooting](https://deepwiki.com/tenstorrent/tt-topology/8-troubleshooting).
