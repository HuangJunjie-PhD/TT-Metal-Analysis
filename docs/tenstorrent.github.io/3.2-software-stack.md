## Purpose and Overview

This page provides a technical overview of the Tenstorrent software stack, which consists of multiple layers that enable developers to program, optimize, and run workloads on Tenstorrent hardware. The stack includes high-level APIs, low-level libraries, system drivers, and management utilities that collectively form a comprehensive development ecosystem for Tenstorrent's Tensix processors [core/getting-started/tt-software-stack.md 7-10](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/getting-started/tt-software-stack.md?plain=1#L7-L10)

The software stack is designed to support a variety of entry points, from high-level model compilation using standard frameworks like PyTorch and JAX to low-level C++ kernel programming for maximum hardware performance [core/getting-started/tt-software-stack.md 72-106](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/getting-started/tt-software-stack.md?plain=1#L72-L106)

## Major Frameworks

The stack is primarily divided into three main components: TT-Forge, TT-NN, and TT-Metalium.

### TT-Forge: The MLIR-Based Compiler

TT-Forge™ is Tenstorrent's Multi-Level Intermediate Representation (MLIR)-based compiler [core/getting-started/tt-software-stack.md 72-74](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/getting-started/tt-software-stack.md?plain=1#L72-L74) It bridges high-level machine learning frameworks with the Tenstorrent software stack, offering an automated path to run models from PyTorch, JAX, and TensorFlow without custom kernel development [core/getting-started/tt-software-stack.md 74-78](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/getting-started/tt-software-stack.md?plain=1#L74-L78)

For details, see [TT-Forge](https://deepwiki.com/tenstorrent/tenstorrent.github.io/3.2.2-tt-forge).

### TT-NN: Neural Network Operation Library

TT-NN™ is a library of neural network operations providing a user-friendly interface for running models. It is designed to be intuitive for developers familiar with PyTorch and provides a stable set of pre-packaged, optimized operations in both Python and C++ [core/getting-started/tt-software-stack.md 82-88](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/getting-started/tt-software-stack.md?plain=1#L82-L88)

For details, see [TT-NN](https://deepwiki.com/tenstorrent/tenstorrent.github.io/3.2.1-tt-nn).

### TT-Metalium: Low-Level SDK

TT-Metalium™ is the low-level, open-source SDK providing direct access to Tenstorrent hardware. It is a bare-metal programming environment for users who need to write custom C++ kernels, manage memory explicitly, or implement novel operations [core/getting-started/tt-software-stack.md 100-106](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/getting-started/tt-software-stack.md?plain=1#L100-L106) It exposes the RISC-V processors and the Network-on-Chip (NoC) within each Tensix core [core/getting-started/tt-software-stack.md 104-106](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/getting-started/tt-software-stack.md?plain=1#L104-L106)

For details, see [TT-Metalium](https://deepwiki.com/tenstorrent/tenstorrent.github.io/3.2.3-tt-metalium).
