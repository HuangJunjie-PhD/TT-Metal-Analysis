TVMScript is a Python-embedded domain-specific language (DSL) for writing TensorIR programs. It provides human-readable, Pythonic syntax for defining tensor computations, schedules, and transformations. TVMScript programs are executable Python code that generate TIR data structures, and they support bidirectional conversion (round-tripping) between Python syntax and TIR representation.

For information about the IR builder API for programmatic construction of TIR, see [IR Builder](https://deepwiki.com/tenstorrent/tt-tvm/5.7-ir-builder). For details on schedule transformation APIs, see [Schedule API](https://deepwiki.com/tenstorrent/tt-tvm/5.3-schedule-api).

## Overview

TVMScript serves as the primary interface for human authorship of TIR programs. It translates Python syntax into TIR data structures at runtime, enabling developers to write low-level tensor programs in a high-level language.

## Core Decorators

TVMScript uses Python decorators to mark functions for parsing and conversion to TIR.

| Decorator | Purpose | Returns |
| --- | --- | --- |
| `@T.prim_func` | Defines a TensorIR primitive function | `tir.PrimFunc` |
| `@tvm.script.ir_module` | Defines an IR module containing multiple functions | `IRModule` |
| `@I.ir_module` | Alternative import for IR module decorator | `IRModule` |

## Block Construct

The `T.block()` construct defines computation blocks with explicit data access patterns.

### Block Components

| Component | Syntax | Purpose |
| --- | --- | --- |
| Block name | `with T.block("name"):` | Identifies the block |
| Axis binding | `T.axis.remap("SS", [i, j])` | Maps loop variables to block axes |
| Read regions | `T.reads(A[vi, vj])` | Declares input data dependencies |
| Write regions | `T.writes(B[vi, vj])` | Declares output data regions |
| Computation | Python statements | Actual computation logic |

### Axis Types

*   `S` (Spatial): Independent iterations, can be parallelized
*   `R` (Reduction): Dependent iterations forming a reduction

## Error Reporting with Span Information

TVMScript preserves source location information through the `Span` system, enabling detailed error messages.

### Error Message Format

When a scheduling error occurs, the error message includes:

1.   The primitive operation that failed
2.   The IR module with diagnostic annotations
3.   Specific locations underlined in the printed IR
4.   A descriptive error message

Example error output structure:

```
ScheduleError: An error occurred in the schedule primitive 'reorder'.

The IR with diagnostic is:
@T.prim_func
def func(...):
    # tir.Block#0
    with T.block("B"):
    ^^^^^^^^^^^^^^^^^^  # This line is underlined

Error message: The input loop {0} of reorder is required to be an ancestor of block {1}.
```

## Validation and Error Checking

TVMScript performs extensive validation during parsing to catch common errors early.

### Common Error Categories

| Error Type | Example | Detection Point |
| --- | --- | --- |
| Missing type annotation | `def func(a):` | Function parameter parsing |
| Invalid buffer binding | `T.match_buffer((16, 16))` | Missing buffer argument |
| Undefined buffer | Reference before declaration | Name resolution |
| Inconsistent axis binding | `vi, vj = T.axis.remap("SS", [i])` | Block axis parsing |
| Invalid loop function | `for i in T.evaluate(0):` | Loop construct validation |
| Duplicate block signature | Multiple `T.reads()` statements | Block signature parsing |
| Non-affine block axes | `T.axis.S(128, i*i)` | Affinity analysis |

## TVMScript vs IRBuilder

| Aspect | TVMScript | IRBuilder |
| --- | --- | --- |
| Interface | Declarative Python syntax | Imperative Python API |
| Use Case | Human authorship, documentation | Programmatic generation, passes |
| Readability | High - looks like Python | Lower - context managers and builders |
| Type checking | Limited (runtime) | Limited (runtime) |
| Round-tripping | Full support via `.script()` | Not applicable |
| Flexibility | Limited to syntax | Full programmatic control |

For details on IRBuilder, see [IR Builder](https://deepwiki.com/tenstorrent/tt-tvm/5.7-ir-builder).
