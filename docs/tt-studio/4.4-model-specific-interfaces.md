This document covers the specialized user interfaces for different AI model types in TT-Studio. These interfaces provide tailored user experiences for object detection, image generation, speech recognition, and text chat models, each optimized for their specific input/output requirements and workflows.

For general chat functionality shared across models, see [Chat UI](https://deepwiki.com/tenstorrent/tt-studio/4.3-chat-ui). For the model deployment and management interfaces, see [Models Deployed Interface](https://deepwiki.com/tenstorrent/tt-studio/4.2-models-deployed-interface).

## Model Type Detection and Routing

TT-Studio automatically determines the appropriate interface for each deployed model based on naming conventions and provides routing to specialized UIs via the `handleModelNavigationClick` function [app/frontend/src/api/modelsDeployedApis.ts 217-234](https://github.com/tenstorrent/tt-studio/blob/a6d347af/app/frontend/src/api/modelsDeployedApis.ts#L217-L234)

## Common Interface Patterns

### State Initialization

All specialized interfaces follow a pattern of extracting the `modelID` (container ID) from the React Router `location.state` to ensure the UI is bound to the correct hardware-accelerated instance:

`// Found in ObjectDetectionComponent, ImageGenParentComponent, and SpeechToTextAppuseEffect(() => {  if (location.state?.containerID) {    setModelID(location.state.containerID);  }}, [location.state]);`
**Sources:**[app/frontend/src/components/object_detection/ObjectDetectionComponent.tsx 103-113](https://github.com/tenstorrent/tt-studio/blob/a6d347af/app/frontend/src/components/object_detection/ObjectDetectionComponent.tsx#L103-L113)[app/frontend/src/components/imageGen/ImageGenParentComponent.tsx 20-31](https://github.com/tenstorrent/tt-studio/blob/a6d347af/app/frontend/src/components/imageGen/ImageGenParentComponent.tsx#L20-L31)[app/frontend/src/components/speechToText/speechToTextApp.tsx 46-57](https://github.com/tenstorrent/tt-studio/blob/a6d347af/app/frontend/src/components/speechToText/speechToTextApp.tsx#L46-L57)
