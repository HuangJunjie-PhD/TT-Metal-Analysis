This page provides detailed information about Tenstorrent's Grayskull™ e75 and e150 Tensix processors, including specifications, installation procedures, and software support details. These processors represent Tenstorrent's first generation of AI accelerator cards, which have since been succeeded by the Wormhole and Blackhole series.

> **Important Note**: Software support for Grayskull has been discontinued. The last supported software versions (TT-KMD 1.31, TT-Metalium v0.55) are listed in the [Software Support](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/Software%20Support) section.

## Processor Models and Specifications

The Grayskull™ architecture is built around a high-density array of Tensix cores, integrated SRAM, and LPDDR4 memory.

| Specification | e75 | e150 |
| --- | --- | --- |
| Part Number | TC-01001 | TC-01002 |
| Tensix Cores | 96 | 120 |
| AI Clock | 1 GHz | 1.2 GHz |
| SRAM | 96 MB | 120 MB |
| Memory | 8 GB LPDDR4 | 8 GB LPDDR4 |
| Memory Speed | 3.2 GT/sec | 3.7 GT/sec |
| Memory Bandwidth | 102 GB/sec | 118 GB/sec |
| TeraFLOPs (FP8) | 221 | 332 |
| TeraFLOPs (FP16) | 55 | 83 |
| TeraFLOPs (BLOCKFP8) | 55 | 83 |
| TBP (Total Board Power) | 75W | 200W |
| External Power | 1x 6-pin PCIe | 1x 6+2-pin PCIe, 1x 6-pin PCIe |
| System Interface | PCI Express 4.0 x16 | PCI Express 4.0 x16 |
| Cooling | Active (Blower) | Passive (Heatsink) |
| Dimensions (w/o Cooling Kit) | 18mm x 167.5mm x 69mm | 36mm x 259.8mm x 111.25mm |
| Dimensions (w/ Cooling Kit) | 18mm x 257mm x 98mm | 36mm x 399mm x 114mm |

## Data Precision Format Support

The e75 and e150 models support a variety of floating point and block floating point formats optimized for AI workloads.

| Format | Abbreviation | Support |
| --- | --- | --- |
| Floating Point (8-bit) | FP8 | Yes |
| Floating Point (16-bit) | FP16 | Yes |
| Brain Floating Point (16-bit) | BF16 | Yes |
| Floating Point (32-bit) | FP32 | No |
| Block Floating Point (2-bit) | BFP2 | Yes |
| Block Floating Point (4-bit) | BFP4 | Yes |
| Block Floating Point (8-bit) | BFP8 | Yes |
| Vector TF19 | VTF19 | Yes |

## Hardware Installation

### Prerequisites

1.   **PCI Express**: A PCIe 4.0 x16 slot is required. For optimal performance, bifurcation should be disabled.
2.   **Power Connectors**: 
    *   **e75**: One 6-pin PCIe power connector (powers the blower fan).
    *   **e150**: One 6+2-pin PCIe and one 6-pin PCIe power connector.

3.   **Environment**: The system must provide ≥30 CFM of airflow if using the e150 without an Active Cooling Kit.

Sources: [core/aibs/grayskull/specifications.md 54-82](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/grayskull/specifications.md?plain=1#L54-L82)[core/aibs/grayskull/installation.md 12-19](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/grayskull/installation.md?plain=1#L12-L19)

### Power Connection Details

*   **e75**: Connect a 6-pin PCIe power cable directly to the fan assembly [core/aibs/grayskull/installation.md 49-53](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/grayskull/installation.md?plain=1#L49-L53)
*   **e150 (Standard)**: Connect a 6+2-pin PCIe cable to the 8-pin port and a 6-pin PCIe cable to the 6-pin port. **Warning**: Do not use 8-pin EPS12V cables [core/aibs/grayskull/installation.md 55-59](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/grayskull/installation.md?plain=1#L55-L59)
*   **e150 (With Active Cooling Kit)**: Connect the 6+2-pin PCIe cable to the card. The 6-pin PCIe cable must connect to the fan harness female plug, with the male harness plug then connecting to the card [core/aibs/grayskull/installation.md 61-65](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/grayskull/installation.md?plain=1#L61-L65)
