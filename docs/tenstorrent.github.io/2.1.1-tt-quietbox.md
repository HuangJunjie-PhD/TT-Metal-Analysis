This document provides detailed specifications, setup procedures, and support information for the Tenstorrent TT-QuietBox™ workstation series. The TT-QuietBox is a liquid-cooled desktop workstation designed for quiet, high-performance AI development, featuring specialized cooling for Tenstorrent Tensix processors.

## Overview

The TT-QuietBox line spans three generations of hardware, each utilizing Tenstorrent's AI accelerators.

1.   **TT-QuietBox Wormhole (TW-04001)**: Features 4x Wormhole n300 processors.
2.   **TT-QuietBox Blackhole (TW-04002)**: Features 4x Blackhole p150c processors.
3.   **TT-QuietBox 2 (QB2) (TW-04003)**: The latest generation, featuring 2x Blackhole p300c cards (4 chips total) and a Ryzen 9000 series host CPU. [core/systems/quietbox/quietbox-bh-2/specifications.md 25-35](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/specifications.md?plain=1#L25-L35)

### Blackhole p300c Card Details (QuietBox 2)

The p300c card (TC-03007) is a specialized dual-ASIC board. Each card contains 240 Tensix cores and 64GB of GDDR6 memory [core/aibs/blackhole/p300.md 27-37](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/p300.md?plain=1#L27-L37) In the QuietBox 2, two of these cards are interconnected via a Samtec ARP6 series high-performance cable [core/systems/quietbox/quietbox-bh-2/specifications.md 65-69](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/specifications.md?plain=1#L65-L69)

## Hardware Setup and Unboxing

### Unboxing (QuietBox 2)

The QuietBox 2 uses a clip-based cardboard packaging system [core/systems/quietbox/quietbox-bh-2/setup.md 52-55](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/setup.md?plain=1#L52-L55)

1.   Remove white side clips at the bottom of the box.
2.   Lift the box lid and remove the accessories box (contains power cord and AnkerWork S500 speakerphone).
3.   Lift the workstation using the handles [core/systems/quietbox/quietbox-bh-2/setup.md 73-75](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/setup.md?plain=1#L73-L75)

### Connectivity and Topology

Each generation requires specific interconnect cabling for the processor mesh:

*   **Wormhole (TW-04001)**: Requires connecting two external QSFP-DD 400GbE cables between Slot 1/4 and Slot 3/2 [core/systems/quietbox/quietbox-wh/setup.md 103-109](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-wh/setup.md?plain=1#L103-L109)
*   **Blackhole (TW-04002)**: Requires eight external QSFP-DD 800GbE cables to create the processor mesh [core/systems/quietbox/quietbox-bh/setup.md 89-91](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh/setup.md?plain=1#L89-L91)
*   **QuietBox 2 (TW-04003)**: Features an **Internal Topology**. The two p300c cards are pre-connected internally via a Samtec ARP6 cable [core/systems/quietbox/quietbox-bh-2/specifications.md 65-69](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/specifications.md?plain=1#L65-L69)

## Maintenance and Support

### Cooling Maintenance

As liquid-cooled systems, the QuietBox units require periodic coolant top-ups, typically **once per year** depending on usage [core/systems/quietbox/quietbox-bh-2/support-bh-2.md 26-28](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/support-bh-2.md?plain=1#L26-L28)

### Safety and Power

*   **QuietBox 2**: Draws up to 1200W during heavy inference. A dedicated 15A circuit is recommended [core/systems/quietbox/quietbox-bh-2/setup.md 22](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/setup.md?plain=1#L22-L22)
*   **QuietBox BH/WH**: Weigh significantly more (80 lbs) and require two people for safe lifting [core/systems/quietbox/quietbox-bh/setup.md 18-20](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh/setup.md?plain=1#L18-L20)

### Troubleshooting

*   **Yellow DRAM Light**: On QuietBox 2, if the system shows a yellow DRAM light on boot, the memory is re-training. Do not power off; this can take up to 20 minutes [core/systems/quietbox/quietbox-bh-2/support-bh-2.md 109-111](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/support-bh-2.md?plain=1#L109-L111)
*   **Welcome Animation**: Can be disabled via `/home/ttuser/scripts/tt-disable-demo-mode.sh`[core/systems/quietbox/quietbox-bh-2/support-bh-2.md 95-99](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/quietbox/quietbox-bh-2/support-bh-2.md?plain=1#L95-L99)
