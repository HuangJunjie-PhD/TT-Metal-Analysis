# r37 Demo Applications

Relevant source files
*   [tensix/whb0/prj/build_demo_llk_r37.sh](https://github.com/tenstorrent/tensix-isa-simulator/blob/afb83529/tensix/whb0/prj/build_demo_llk_r37.sh)

This page documents the demonstration applications for the r37 architecture components of the Tensix ISA Simulator. These applications provide practical examples of how to use the r37 system components, particularly focusing on the Instruction Set Simulator (ISS) and its integration with other libraries.

For information about general LLK demonstration applications not specific to r37, see [LLK Demo Applications](https://deepwiki.com/tenstorrent/tensix-isa-simulator/6.1-llk-demo-applications).

## Overview

The r37 demo applications showcase the practical usage of the r37 architecture components within the Tensix ISA Simulator framework. Currently, the main r37 demo application is `demo_llk_r37`, which demonstrates the interaction between the r37 Instruction Set Simulator and supporting libraries.

## demo_llk_r37 Application

The `demo_llk_r37` application demonstrates how to use the r37 Instruction Set Simulator in conjunction with various supporting libraries to create a functional simulation environment for the r37 architecture.

### Build Configuration

The application is built using a dedicated build script:

[tensix/whb0/prj/build_demo_llk_r37.sh 12-24](https://github.com/tenstorrent/tensix-isa-simulator/blob/afb83529/tensix/whb0/prj/build_demo_llk_r37.sh#L12-L24)

This script compiles all source files in the `demo/llk_r37` directory and links them with several essential libraries to create the executable.

### Dependencies

The demo application depends on the following libraries:

| Library | Description |
| --- | --- |
| r37_iss.a | r37 Instruction Set Simulator library - provides the core simulation functionality for the r37 architecture |
| core.a | Core utility library - provides fundamental data structures and functions |
| schedule.a | Scheduling library - manages task scheduling and execution |
| sfpi.a | Serial Flash Programming Interface library - handles flash memory programming operations |

Sources: [tensix/whb0/prj/build_demo_llk_r37.sh 21-24](https://github.com/tenstorrent/tensix-isa-simulator/blob/afb83529/tensix/whb0/prj/build_demo_llk_r37.sh#L21-L24)

## Building and Running

To build the demo application:

1.   Navigate to the project directory:

`cd tensix/whb0/prj`
2.   Run the build script:

`./build_demo_llk_r37.sh`
3.   The executable will be created at `../bin/demo_llk_r37`

Sources: [tensix/whb0/prj/build_demo_llk_r37.sh 9-10](https://github.com/tenstorrent/tensix-isa-simulator/blob/afb83529/tensix/whb0/prj/build_demo_llk_r37.sh#L9-L10)

## Practical Applications

The `demo_llk_r37` application serves several practical purposes:

1.   **Learning Resource**: Provides a concrete example of how to use the r37 ISS
2.   **Integration Example**: Demonstrates how to integrate the r37 ISS with supporting libraries
3.   **Testing Tool**: Can be used to test r37 architecture components
4.   **Reference Implementation**: Serves as a reference for developing applications that use the r37 architecture

## Conclusion

The r37 demo applications, particularly `demo_llk_r37`, provide valuable examples of how to use the r37 architecture components within the Tensix ISA Simulator. These demos illustrate the integration between the Instruction Set Simulator and supporting libraries, serving as reference implementations for developers working with the r37 architecture.

For information about other demonstration applications in the Tensix ISA Simulator, see [Demo Applications](https://deepwiki.com/tenstorrent/tensix-isa-simulator/6-demo-applications).
