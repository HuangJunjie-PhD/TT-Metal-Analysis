This document provides detailed information about Tenstorrent's Blackhole p100a, p150a, and p150b Tensix Processor add-in boards, including technical specifications, installation procedures, connectivity options, and system requirements.

## Overview

Blackhole processors represent Tenstorrent's latest generation Tensix Processor technology, available as PCIe add-in boards in three models: p100a, p150a, and p150b. These processors are designed for high-performance AI workloads with powerful tensor compute capabilities and significant on-chip memory.

Sources: [core/aibs/blackhole/specifications.md 13-20](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/specifications.md?plain=1#L13-L20)[core/aibs/blackhole/support.md 12-16](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L12-L16)

## Technical Specifications

### Processor Architecture

The Blackhole Tensix Processor architecture features a high-density grid of Tensix cores complemented by "Big RISC-V" cores for complex control logic.

Sources: [core/aibs/blackhole/specifications.md 13-20](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/specifications.md?plain=1#L13-L20)[core/aibs/blackhole/specifications.md 33-36](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/specifications.md?plain=1#L33-L36)

## Physical Installation

### Installation Process

The physical installation involves standard PCIe card procedures with specific attention to power and airflow requirements.

Sources: [core/aibs/blackhole/installation.md 14-47](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L14-L47)[core/aibs/blackhole/support.md 46-49](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L46-L49)

### Pre-Installation Requirements

1.   **Disconnect power** to the host computer prior to installation. [core/aibs/blackhole/installation.md 18](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L18-L18)
2.   Verify that the system provides: 
    *   **PCI Express 5.0 x16 slot**: For optimal performance, the card requires a x16 configuration without bifurcation. [core/aibs/blackhole/installation.md 20](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L20-L20)
        *   The p100a and p150a are dual-slot width cards with active coolers; leave the adjacent slot unoccupied for airflow. [core/aibs/blackhole/installation.md 21](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L21-L21)
        *   The p150b is a dual-slot width card with a passive heatsink intended for rack-mounted systems with forced air cooling. [core/aibs/blackhole/installation.md 22](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L22-L22)

    *   **ATX 3.1 Certified power supply** or better; using an older or inadequate power supply may result in system instability. [core/aibs/blackhole/installation.md 23-25](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L23-L25)
    *   One (1) **12+4-pin 12V-2x6 power connector**. [core/aibs/blackhole/installation.md 26](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L26-L26)

3.   Discharge your body's static electricity by wearing an **ESD wrist strap** or touching a grounded surface. [core/aibs/blackhole/installation.md 27](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L27-L27)

### Installation Steps

1.   Insert the card into the **PCIe x16 slot** and secure it with the necessary screws. [core/aibs/blackhole/installation.md 35](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L35-L35)
2.   Connect a **12+4-pin 12V-2x6 power cable** to the plug on the back of the card. [core/aibs/blackhole/installation.md 41](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L41-L41)
3.   Ensure the cable is fully and securely connected to prevent instability or damage. [core/aibs/blackhole/installation.md 42-43](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/installation.md?plain=1#L42-L43)

## Connectivity

### QSFP-DD Ports (p150a/p150b)

The p150a and p150b models feature four ports for high-speed inter-card networking.

Each of the four passive QSFP-DD ports provides **800 Gbps connectivity** and supports cable lengths up to 2 meters (6.5 feet). These ports connect exclusively to other Blackhole™-based add-in cards. [core/aibs/blackhole/specifications.md 62-68](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/specifications.md?plain=1#L62-L68)

Validated cable models include:

*   Amphenol 0.5m/1m 800G Passive QSFP-DD [core/aibs/blackhole/support.md 22-23](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L22-L23)
*   FS 0.5m/1m 800G Passive QSFP-DD [core/aibs/blackhole/support.md 24-25](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L24-L25)

## Data Precision Formats

The Blackhole™ Tensix Processor supports a wide range of precision formats for both tensor and scalar operations.

| Format | Bit Depth (Tensix Cores) | Bit Depth (Big RISC-V Cores) |
| --- | --- | --- |
| Floating point | FP8, FP16, BFLOAT16 FP32 (Output Only) | FP8, FP16, BFLOAT16, FP32, FP64 |
| Block floating point | BLOCKFP2, BLOCKFP4, BLOCKFP8 | - |
| Integer | INT8 INT32 (Output Only) | INT8, INT16, INT32, INT64 |
| Unsigned Integer | UINT8 | - |
| TensorFloat | TF32 | - |
| Vector | VTF19, VFP32 | VFP64 |

## Troubleshooting

*   **Enumeration Issues**: If the card is not detected, manually force PCIe operation to `Gen 4.0` or `Gen 5.0` in the system **BIOS** instead of using "Auto". [core/aibs/blackhole/support.md 46-49](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L46-L49)
*   **Idle Power**: Blackhole processors typically consume ~120W at idle. [core/aibs/blackhole/support.md 27-29](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L27-L29)
*   **Thermal Management**: The processor is designed to operate safely up to **95°C**. The card will automatically throttle clock speeds to maintain safe operation. [core/aibs/blackhole/support.md 33-35](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L33-L35)
*   **Firmware**: Once enumerated, ensure you update to the latest firmware using the Tenstorrent software stack tools. [core/aibs/blackhole/support.md 50-52](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L50-L52)
