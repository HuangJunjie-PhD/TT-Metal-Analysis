## Purpose and Scope

This document provides instructions for installing the `tt-perf-report` tool, which analyzes performance traces from TT-Metal operations. The tool helps identify performance bottlenecks and provides optimization advice for ML workloads running on Tenstorrent hardware. For information about using the tool after installation, see [Usage Guide](https://deepwiki.com/tenstorrent/tt-perf-report/3-usage-guide). For generating traces suitable for analysis, see [Generating Traces](https://deepwiki.com/tenstorrent/tt-perf-report/3.2-generating-traces).

## Prerequisites

Before installing tt-perf-report, ensure your system meets the following requirements:

*   Python 3.6 or higher
*   pip (Python package installer)
*   For recommended installation: pipx

The only direct dependency required is pandas, as specified in the project configuration.

## Troubleshooting Common Issues

### Command Not Found

If you encounter a "command not found" error:

*   Check that the Python scripts directory is in your PATH
*   For pipx installations, run `pipx ensurepath`
*   Try reopening your terminal session

### Dependency Issues

If you encounter dependency-related errors:

*   Ensure pandas is installed (`pip install pandas`)
*   Check your Python version: `python --version`
