This page provides a high-level overview of Tenstorrent's complete system offerings, including pre-configured workstations and servers integrated with Tenstorrent Tensix processors. These systems are designed for AI/ML development, fine-tuning, and inference, providing a turnkey experience for the Tenstorrent software stack.

## System Portfolio Overview

Tenstorrent systems are categorized by their processor generation (Wormhole or Blackhole) and form factor (Desktop Workstation or Rack-mount Server).

Sources: [core/systems/index.rst 1-21](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/index.rst#L1-L21)[core/README.md 18-32](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/README.md?plain=1#L18-L32)

## Product Lines

### TT-QuietBox

The TT-QuietBox is a premium, liquid-cooled workstation designed for office environments where low noise is critical. It supports both Wormhole and Blackhole processor generations.

*   **Wormhole Version (TW-04001):** Features 4x n300 processors and 512GB DDR5 memory.
*   **Blackhole Version (TW-04002):** Features 4x p150c processors and 256GB DDR5 memory.
*   **QB2:** The next-generation liquid-cooled workstation.

For details, see [TT-QuietBox](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.1.1-tt-quietbox).

### TT-LoudBox (T3000)

The TT-LoudBox is a 4U system capable of acting as a high-performance desktop or a rack-mounted server. It is based on the SuperMicro SYS-740GP-TNRT chassis [core/systems/t3000/specifications.md 24](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t3000/specifications.md?plain=1#L24-L24)

*   **TW-02001:** Equipped with 4x Wormhole n150s processors.
*   **TW-02002:** Equipped with 4x Wormhole n300s processors and internal Warp 100 bridges for mesh connectivity [core/systems/t3000/specifications.md 19](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t3000/specifications.md?plain=1#L19-L19)

For details, see [TT-LoudBox](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.1.2-tt-loudbox).

### T7000 Server

The T7000 (TW-03001) is a high-density 4U rack-mount server featuring 8x Wormhole n150s Tensix processors [core/systems/t7000/specifications.md 13-18](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t7000/specifications.md?plain=1#L13-L18) It utilizes a dual AMD EPYC™ 7352 host configuration and is designed for large-scale model execution [core/systems/t7000/specifications.md 15](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t7000/specifications.md?plain=1#L15-L15)

For details, see [T7000](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.1.3-t7000).

### T1000 Workstation

The T1000 (TW-01001) is a compact, entry-level workstation featuring a single Wormhole n150s Tensix processor [core/systems/t1000/specifications.md 12-17](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t1000/specifications.md?plain=1#L12-L17) It is based on the SuperMicro SYS-E403-12P-FN2T chassis and is ideal for developers starting with the Tenstorrent architecture [core/systems/t1000/specifications.md 21](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t1000/specifications.md?plain=1#L21-L21)

For details, see [T1000](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.1.4-t1000).

## System Architecture and Topology

Tenstorrent systems utilize high-speed interconnects (Warp 100 and QSFP-DD) to create a unified compute mesh. The software manages these connections via the `tt-topology` utility [core/systems/t3000/specifications.md 57](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t3000/specifications.md?plain=1#L57-L57)
