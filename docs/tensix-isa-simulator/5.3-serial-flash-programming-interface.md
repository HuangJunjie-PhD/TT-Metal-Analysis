The Serial Flash Programming Interface (SFPI) library provides functionality for interacting with serial flash memory within the Tensix ISA Simulator. It enables simulation and programming of serial flash memory devices, which are commonly used in embedded systems for storing firmware, configuration data, and other non-volatile information.

## Build System

The SFPI library and its demo application are built using dedicated build scripts:

### SFPI Library Build

The SFPI library is built using the `build_sfpi.sh` script, which compiles all C++ source files in the `src/sfpi/` directory and creates a static library `sfpi.a` in the `lib/` directory.

### Demo Application Build

The SFPI demo application (`demo_sfpi_gen`) is built using the `build_demo_sfpi_gen.sh` script, which compiles all C++ source files in the `src/demo/sfpi_gen/` directory, links them with the SFPI library, and creates an executable in the `bin/` directory.

## Memory Organization

Serial flash memory is typically organized in a hierarchical structure:

| Level | Description | Typical Size | Operations |
| --- | --- | --- | --- |
| Page | Smallest programmable unit | 256B - 4KB | Read, Write |
| Sector | Group of pages | 4KB - 64KB | Erase, Read |
| Block | Group of sectors | 64KB - 256KB | Erase, Read |
| Chip | Complete memory | 1MB - 16MB+ | Chip Erase, Read |

The SFPI library likely provides abstractions for managing this organization, allowing operations at different levels of the hierarchy.

## Application Scenarios

The SFPI library can be applied in various scenarios within the Tensix ISA Simulator:

1.   **Firmware Simulation**: Simulating firmware loading and execution from flash memory
2.   **Configuration Management**: Handling device configuration stored in flash memory
3.   **Testing and Validation**: Verifying system behavior with different flash memory states
4.   **Performance Analysis**: Analyzing system performance when interacting with flash memory

These scenarios highlight the importance of the SFPI library in providing realistic simulation of flash memory behavior in the Tensix ISA Simulator environment.
