## Purpose and Scope

Model Explorer is a hierarchical graph visualization tool that provides intuitive exploration and debugging capabilities for model graphs within the Tenstorrent ecosystem. This system enables developers to dynamically examine model structures through expandable/collapsible layers, overlay metadata, and perform interactive analysis of neural network architectures.

For information about analyzing model execution and performance, see [TT-NN Visualizer](https://deepwiki.com/tenstorrent/tt-tutorial/6.2-tt-nn-visualizer). For comprehensive model development and optimization workflows, see [Model Tutorials and Bring-up](https://deepwiki.com/tenstorrent/tt-tutorial/5.1-model-tutorials-and-bring-up).

## Key Capabilities

### Hierarchical Graph Organization

Model Explorer organizes model operations into nested hierarchical layers, enabling users to:

*   **Dynamic Layer Control**: Expand or collapse operational layers to focus on specific model components
*   **Nested Structure Navigation**: Navigate through complex model architectures with intuitive layer-based organization
*   **Interactive Layer Exploration**: Use pop-up displays to examine detailed layer information without losing context

### Visualization and Analysis Features

| Feature | Description | Use Case |
| --- | --- | --- |
| Input/Output Highlighting | Visual emphasis on data flow entry and exit points | Debugging data flow issues |
| Metadata Overlay | Custom data annotations on graph nodes | Performance metrics, debugging info |
| Interactive Pop-ups | Detailed layer information display | In-depth component analysis |
| Search Functionality | Node and operation discovery | Large model navigation |
| Identical Layer Detection | Pattern recognition across model structure | Optimization opportunity identification |
| GPU-Accelerated Rendering | High-performance graph visualization | Real-time interaction with large models |
