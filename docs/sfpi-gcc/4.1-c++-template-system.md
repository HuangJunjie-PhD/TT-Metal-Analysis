## Purpose and Scope

The C++ Template System in sfpi-gcc implements the complete C++ template processing infrastructure, including template instantiation, specialization, argument deduction, and C++20 concepts. This system handles the transformation of template declarations into concrete instantiated code during compilation.

For information about general GCC optimization passes, see [3.3](https://deepwiki.com/tenstorrent/sfpi-gcc/3.3-pass-management-and-optimization-pipeline). For details about language frontend integration, see [4](https://deepwiki.com/tenstorrent/sfpi-gcc/4-language-support).
