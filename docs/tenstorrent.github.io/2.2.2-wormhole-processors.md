This document provides comprehensive information about Tenstorrent's Wormhole™ series Tensix processors, including the n150d, n150s, n300d, and n300s models. It covers specifications, features, hardware installation, and integration with Tenstorrent's software ecosystem. For information about other Tensix processors, see [Blackhole Processors](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.2.1-blackhole-processors) or [Grayskull Processors](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.2.3-grayskull-processors).

## Overview

Wormhole processors are Tenstorrent's second-generation AI accelerator add-in boards (AIBs) designed for high-performance machine learning workloads. These processors feature improved architecture, memory bandwidth, and connectivity options compared to the earlier Grayskull series.

The Wormhole family includes four primary models:

*   **n150s**: Single-ASIC passive-cooled card (160W) [core/aibs/wormhole/specifications.md 15-34](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L15-L34)
*   **n150d**: Single-ASIC active-cooled card (160W) [core/aibs/wormhole/specifications.md 15-34](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L15-L34)
*   **n300s**: Dual-ASIC passive-cooled card (300W) [core/aibs/wormhole/specifications.md 15-34](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L15-L34)
*   **n300d**: Dual-ASIC active-cooled card with axial fan design (300W) [core/aibs/wormhole/specifications.md 15-34](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L15-L34)

## Hardware Architecture

Wormhole processors feature Tenstorrent's proprietary Tensix architecture. Each Wormhole ASIC contains 80 Tensix cores (with a subset enabled depending on the model) and 120 MB of on-chip SRAM [core/aibs/wormhole/specifications.md 5-9](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L5-L9)

## Hardware Installation

### Pre-Installation Requirements

Before installing a Wormhole processor, ensure your system meets these requirements:

1.   **PCIe 4.0 x16 slot**: For optimal performance without bifurcation [core/aibs/wormhole/installation.md 17](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L17-L17)
2.   **EPS12V 4+4-pin power connector**: Do NOT use a 6+2-pin PCIe power cable [core/aibs/wormhole/installation.md 51](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L51-L51)
3.   **Cooling considerations**: 
    *   **n150s/n300s**: Passive-cooled. Require the [Active Cooling Kit](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.2.4-accessories) for desktop workstations [core/aibs/wormhole/specifications.md 13](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L13-L13)
    *   **n150d/n300d**: Active-cooled with axial fans. Ensure sufficient chassis airflow [core/aibs/wormhole/installation.md 19](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L19-L19)

4.   **Memory**: Minimum 64 GB system RAM [core/aibs/wormhole/specifications.md 73](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L73-L73)

Sources: [core/aibs/wormhole/installation.md 12-21](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L12-L21)[core/aibs/wormhole/specifications.md 66-81](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L66-L81)

### Installation Procedure

The installation process differs slightly between workstation and server environments:

#### Desktop Workstation Installation

1.   Insert the card into the PCIe x16 slot [core/aibs/wormhole/installation.md 29](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L29-L29)
2.   Secure with necessary screws.
3.   Connect power using the EPS12V cable [core/aibs/wormhole/installation.md 51](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L51-L51)

#### Server Installation (n150s/n300s Only)

1.   House the card in the server casing/housing [core/aibs/wormhole/installation.md 39](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L39-L39)
2.   Lower the encased card into the chassis [core/aibs/wormhole/installation.md 45](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L45-L45)
3.   Secure with required screws and connect power.

Sources: [core/aibs/wormhole/installation.md 23-54](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L23-L54)

### Power Connection

Connect an **EPS12V 4+4-pin power cable** to the **8-pin plug** on the card [core/aibs/wormhole/installation.md 51](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L51-L51)**WARNING:** Do not connect a 6+2-pin PCIe power cable; the pinouts are incompatible and can cause damage [core/aibs/wormhole/installation.md 51](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/installation.md?plain=1#L51-L51)

1.   **2x Warp 100 Bridge Connectors**: Internal notches for direct card-to-card communication [core/aibs/wormhole/specifications.md 49](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L49-L49)
