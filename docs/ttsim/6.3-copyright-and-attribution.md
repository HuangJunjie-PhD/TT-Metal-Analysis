## Attribution Requirements

The Apache License 2.0 Section 4 specifies the attribution requirements for redistribution. When distributing ttsim or derivative works, you must comply with the following requirements:

### Required Elements

#### 1. License Copy

You must provide recipients with a copy of the Apache License 2.0.

**Sources:**[LICENSE 94-95](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L94-L95)

#### 2. Modification Notices

If you modify any files, you must include prominent notices stating that you changed the files.

**Sources:**[LICENSE 97-98](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L97-L98)

#### 3. Copyright and Attribution Notices

You must retain all copyright, patent, trademark, and attribution notices from the original Source form in any derivative works you distribute, excluding notices that do not pertain to any part of the derivative works.

**Sources:**[LICENSE 100-104](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L100-L104)

#### 4. NOTICE File Contents

If the Work includes a NOTICE text file (which ttsim does), any derivative works you distribute must include a readable copy of the attribution notices contained within that NOTICE file in at least one of these locations:

*   Within a NOTICE text file distributed as part of the derivative works
*   Within the Source form or documentation
*   Within a display generated by the derivative works (where third-party notices normally appear)

**Sources:**[LICENSE 106-115](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L106-L115)

## Attribution in Derivative Works

### Additional Attribution Notices

You may add your own attribution notices within derivative works alongside or as an addendum to the NOTICE text from the Work, provided that such additional attribution notices cannot be construed as modifying the License.

The NOTICE file contents are for informational purposes only and do not modify the License terms. They serve to inform users about the origins and copyright of the software.

**Sources:**[LICENSE 116-117](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L116-L117)

## Practical Compliance Examples

### Binary Distribution

When distributing the ttsim binary (`libttsim_wh.so`, `libttsim_bh.so`):

| Required Element | Implementation |
| --- | --- |
| LICENSE file | Include `LICENSE` file in distribution package |
| NOTICE file | Include `NOTICE` file in distribution package |
| Copyright notices | Preserved in binary (if present in source) |

### Source Distribution

When distributing modified ttsim source code:

| Required Element | Implementation |
| --- | --- |
| LICENSE file | Include `LICENSE` file at repository root |
| NOTICE file | Include `NOTICE` file at repository root |
| Copyright notices | Retain all original `Copyright 2025 - Tenstorrent AI ULC` notices in source files |
| Modification notices | Add comments to modified files: "Modified by [Your Name] on [Date]" |
| Change documentation | Document what you changed and why |

### Documentation Distribution

When distributing documentation derived from ttsim documentation:

| Required Element | Implementation |
| --- | --- |
| NOTICE contents | Include attribution notice in documentation footer or credits section |
| LICENSE reference | Link to or include Apache License 2.0 |
| Origin description | State that documentation is derived from ttsim by Tenstorrent AI ULC |

**Sources:**[LICENSE 89-122](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L89-L122)

## Attribution File Locations

The ttsim repository contains two primary legal files at the root:

*   **LICENSE**: Full text of Apache License 2.0 (201 lines)
*   **NOTICE**: Copyright and attribution statement (16 lines)

Both files must be distributed together with any ttsim binaries or source code distributions.

**Sources:**[NOTICE 1-16](https://github.com/tenstorrent/ttsim/blob/67b1733e/NOTICE#L1-L16)[LICENSE 1-202](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L1-L202)

## Copyright Statement Format

When referencing ttsim in your own documentation or software, use this format:

```
This product includes ttsim, software developed at Tenstorrent AI ULC
(https://tenstorrent.com/), licensed under the Apache License, Version 2.0.

Copyright 2025 - Tenstorrent AI ULC
```

This format mirrors the structure in the NOTICE file and satisfies the attribution requirements.

**Sources:**[NOTICE 1-3](https://github.com/tenstorrent/ttsim/blob/67b1733e/NOTICE#L1-L3)

## Relationship to Licensing Terms

The copyright ownership by Tenstorrent AI ULC is separate from the license grants:

*   **Copyright**: Establishes Tenstorrent AI ULC as the rights holder
*   **License**: Grants specific permissions to use the software under Apache 2.0 terms
*   **Attribution**: Requires acknowledgment of Tenstorrent's authorship

For details on what rights are granted, see [Licensing](https://deepwiki.com/tenstorrent/ttsim/6.2-licensing).
