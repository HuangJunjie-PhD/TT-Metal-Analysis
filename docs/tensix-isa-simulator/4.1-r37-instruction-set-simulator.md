## Purpose and Overview

The r37 Instruction Set Simulator (ISS) is a specialized component in the Tensix ISA Simulator that provides an accurate software simulation of the r37 architecture instruction set. It enables developers to execute and test r37 code without requiring physical hardware. The ISS serves as a foundational component for other r37 system elements, including the r37 LLK implementation (see [r37 LLK Implementation](https://deepwiki.com/tenstorrent/tensix-isa-simulator/4.2-r37-llk-implementation)) and compute components (see [r37 Compute Components](https://deepwiki.com/tenstorrent/tensix-isa-simulator/4.3-r37-compute-components)).

Sources: [tensix/whb0/prj/build_r37_iss.sh](https://github.com/tenstorrent/tensix-isa-simulator/blob/afb83529/tensix/whb0/prj/build_r37_iss.sh)[tensix/whb0/prj/build_demo_llk_r37.sh](https://github.com/tenstorrent/tensix-isa-simulator/blob/afb83529/tensix/whb0/prj/build_demo_llk_r37.sh)

## Build System and Components

The r37 ISS is compiled into a static library (`r37_iss.a`) using the build script in the repository. This script compiles C++ source files from the `src/r37/iss/` directory into the library.

### r37 ISS Build Structure

Sources: [tensix/whb0/prj/build_r37_iss.sh 9-16](https://github.com/tenstorrent/tensix-isa-simulator/blob/afb83529/tensix/whb0/prj/build_r37_iss.sh#L9-L16)

The build process includes several key include directories:

| Include Directory | Purpose |
| --- | --- |
| `src/r37/iss` | Main ISS implementation files |
| `src/r37/common/inc` | Common headers for r37 architecture |
| `src/r37/llk_lib` | LLK library interfaces |
| `src/r37/etc` | General utility headers |
| `src/r37/etc/defines` | Architecture-specific definitions |

### r37 ISS Position in System Architecture

Key integration points:

1.   The ISS forms the foundation of the r37 System architecture
2.   Higher-level components depend on the ISS: 
    *   r37 LLK implementation relies on ISS for instruction execution
    *   r37 compute kernel uses ISS for computational instruction simulation
    *   r37 compute library builds upon ISS capabilities

## Usage Example: Demo Application

The repository includes a demo application (`demo_llk_r37`) that demonstrates the practical use of the r37 ISS. This application is built using the script at `tensix/whb0/prj/build_demo_llk_r37.sh`.

### Demo Application Dependencies

Sources: [tensix/whb0/prj/build_demo_llk_r37.sh 12-24](https://github.com/tenstorrent/tensix-isa-simulator/blob/afb83529/tensix/whb0/prj/build_demo_llk_r37.sh#L12-L24)

The demo application links against several libraries:

| Library | Purpose |
| --- | --- |
| `r37_iss.a` | The r37 Instruction Set Simulator |
| `core.a` | Core functionality library |
| `schedule.a` | Task scheduling library |
| `sfpi.a` | Serial Flash Programming Interface library |

The demo source files are located in the `src/demo/llk_r37/` directory and provide a reference implementation showing how to integrate the r37 ISS with other libraries to create a functional r37 simulation environment.
