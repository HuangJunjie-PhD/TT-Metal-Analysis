This page explains the legal framework governing the use, modification, and redistribution of ttsim software. ## Overview

The ttsim repository operates under a dual-scope licensing model:

1.   **Software License**: Apache License 2.0 governs the simulator software itself
2.   **Hardware/IP License**: Separate licensing may be required for hardware-related activities

This distinction is explicitly documented to prevent confusion about what rights are granted when using ttsim.

## Hardware and IP Licensing: Critical Distinction

The most important aspect of ttsim licensing is understanding what is **NOT** covered by the Apache 2.0 license:

### LICENSE_understanding.txt Clarification

The [LICENSE_understanding.txt 1-4](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE_understanding.txt#L1-L4) file provides explicit clarification:

> "For the avoidance of doubt, this software assists in programming Tenstorrent products.
> 
> 
> However, making, using, or selling hardware, models, or IP may require the license of rights (such as patent rights) from Tenstorrent or others."

This means:

| Activity | Covered by Apache 2.0? | Notes |
| --- | --- | --- |
| Using ttsim to develop software | ✓ Yes | Fully permitted |
| Running TT-Metalium workloads on ttsim | ✓ Yes | This is the intended use |
| Modifying ttsim and redistributing it | ✓ Yes | With attribution requirements |
| Studying how Tenstorrent chips work via simulation | ✓ Yes | Educational and development use |
| Manufacturing Wormhole/Blackhole compatible chips | ✗ No | Requires separate hardware license |
| Creating RTL implementations of simulated components | ✗ No | IP rights not granted |
| Selling products containing Tenstorrent IP | ✗ No | Patent/IP license required |
| Using knowledge from ttsim in competing hardware | ✗ No | IP protection applies |

**Sources:**[LICENSE_understanding.txt 1-4](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE_understanding.txt#L1-L4)

## Patent License Terms

The Apache License 2.0 includes a patent license grant with specific scope and termination conditions:

### Scope of Patent Grant

Patent rights are granted **only** for:

*   Patent claims necessarily infringed by the contribution itself
*   Patent claims necessarily infringed by combining the contribution with the Work

This is a defensive patent license intended to allow use of the software, not to grant general rights to Tenstorrent's patent portfolio [LICENSE 73-87](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L73-L87)

### Patent License Termination

The patent license automatically terminates if you:

*   File patent litigation against any entity
*   Allege that the Work or any contribution constitutes patent infringement
*   File cross-claims or counterclaims related to patents

This creates a defensive shield: users who sue over patents lose their patent rights to use the software [LICENSE 82-87](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L82-L87)

**Sources:**[LICENSE 73-87](https://github.com/tenstorrent/ttsim/blob/67b1733e/LICENSE#L73-L87)
