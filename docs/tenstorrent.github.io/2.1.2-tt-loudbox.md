The TT-LoudBox series represents Tenstorrent's high-performance workstation and server line, designed for intensive AI/ML workloads. This page covers both the **Wormhole-based T3000** and the **Blackhole-based LoudBox**, including specifications, rack installation, mesh topology cabling, and multi-node scaling.

## System Overview

The TT-LoudBox line spans two generations of Tenstorrent hardware:

1.   **TT-LoudBox (Wormhole) / T3000**: A 4U/Desktop workstation based on the SuperMicro SYS-740GP-TNRT platform, featuring Wormhole n150s or n300s processors. [core/systems/t3000/specifications.md 5-24](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t3000/specifications.md?plain=1#L5-L24)
2.   **TT-LoudBox (Blackhole)**: A high-density 4U server based on the Supermicro AS-4125GS-TNRT platform, featuring eight Blackhole p150b processors. [core/systems/loudbox-bh/specifications.md 13-43](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/loudbox-bh/specifications.md?plain=1#L13-L43)

## TT-LoudBox (Blackhole) Specifications

The Blackhole-based LoudBox is a pre-launch high-performance server.

| Specification | Details |
| --- | --- |
| **Form Factor** | 4U Rackmount; 441 x 179 x 736 mm (WxHxD). Weight: 45kg (99lbs) |
| **CPU** | 2x AMD EPYC 9124 |
| **Memory** | 768 GB (24x32 GB) DDR5 ECC RDIMM |
| **Tensix Processors** | 8x Tenstorrent Blackhole™ p150b Tensix Processors |
| **Processor Connectivity** | 32x QSFP-DD Passive 800G (4x per card) |
| **Host Connectivity** | 2x RJ45 10GbE, 1x Dedicated BMC, 2x QSFP56 100GbE (NIC) |
| **Power Supply** | 2+2 2000W Titanium PSUs (4kW peak consumption) |
| **Base System** | Supermicro GPU A+ Server AS-4125GS-TNRT |

Sources: [core/systems/loudbox-bh/specifications.md 30-43](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/loudbox-bh/specifications.md?plain=1#L30-L43)

### Setup and Installation

**Safety Warning:** The server weighs 45kg (99lbs). At least two people are required for unboxing and installation. [core/systems/loudbox-bh/setup.md 25-28](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/loudbox-bh/setup.md?plain=1#L25-L28)

1.   **Unboxing**: Remove the four clips from the shipping box, lift the top, and remove the rack rails. [core/systems/loudbox-bh/setup.md 72-84](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/loudbox-bh/setup.md?plain=1#L72-L84)
2.   **Rack Mounting**: Install inner rails onto the chassis and outer rails onto the rack (supports 26.5" to 36.4" depth). [core/systems/loudbox-bh/setup.md 115-157](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/loudbox-bh/setup.md?plain=1#L115-L157)
3.   **OS Installation**: Install Ubuntu 22.04 (Jammy Jellyfish). [core/systems/loudbox-bh/OS-setup.md 20-22](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/loudbox-bh/OS-setup.md?plain=1#L20-L22)
4.   **Verification**: Use `lspci -d 1e52:` to verify all eight accelerators are recognized. [core/systems/loudbox-bh/OS-setup.md 53-73](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/loudbox-bh/OS-setup.md?plain=1#L53-L73)

## Mesh Topology and Scaling

Tenstorrent systems use QSFP-DD cables to create a mesh network between Tensix processors, allowing for low-latency data movement across cards and nodes.

### Internal T3000 (TW-02002) Topology

The T3000 ships with internal Warp 100 bridges. Users must connect the external QSFP-DD cables to complete the local mesh:

*   **Cable 1**: Port 1 on Slot 1 <--> Port 1 on Slot 4. [core/systems/t3000/specifications.md 51](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t3000/specifications.md?plain=1#L51-L51)
*   **Cable 2**: Port 2 on Slot 3 <--> Port 2 on Slot 2. [core/systems/t3000/specifications.md 53](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t3000/specifications.md?plain=1#L53-L53)
