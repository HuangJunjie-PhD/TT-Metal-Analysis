## Purpose and Scope

This guide provides practical examples for using vLLM's Python API and CLI to perform inference with language models. It covers basic text generation, chat completion, embeddings, and multi-modal inputs.

For installation instructions, see [Installation Guide](https://deepwiki.com/tenstorrent/vllm/1.1-installation-guide). For a list of supported model architectures, see [Supported Models](https://deepwiki.com/tenstorrent/vllm/1.2-supported-models). For details on the engine architecture and request processing, see [Core Architecture](https://deepwiki.com/tenstorrent/vllm/2-core-architecture).

## Sampling Parameters

Control generation behavior using `SamplingParams`:

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `temperature` | float | 1.0 | Randomness (0 = greedy, higher = more random) |
| `top_p` | float | 1.0 | Nucleus sampling cumulative probability |
| `top_k` | int | 0 | Top-k sampling (0 = disabled) |
| `max_tokens` | int | 16 | Maximum tokens to generate |
| `stop` | str or list[str] | None | Stop strings |
| `stop_token_ids` | list[int] | None | Stop token IDs |
| `presence_penalty` | float | 0.0 | Penalize repeated tokens |
| `frequency_penalty` | float | 0.0 | Penalize token frequency |
| `n` | int | 1 | Number of completions per prompt |
