## Build Process

The Schedule Library is built using the `build_schedule.sh` script, which compiles the source files and creates a static library.

The build process consists of:

1.   **Environment Setup**: The script defines environment variables for source (`SRC=../src`) and library (`LIB=../lib`) directories.

2.   **Compilation**: All C++ source files in the schedule directory are compiled using g++ with:

    *   C++17 standard (`-std=c++17`)
    *   O3 optimization level (`-O3`)
    *   Source directory include path (`-I $SRC`)

3.   **Library Directory Creation**: The script ensures the library directory exists (`mkdir -p $LIB`).

4.   **Static Library Creation**: Compiled object files are archived into a static library (`ar rsc $LIB/$NAME.a *.o`).

5.   **Cleanup**: Temporary object files are removed (`rm *.o`).
