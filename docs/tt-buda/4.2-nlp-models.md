This document provides detailed information about Natural Language Processing (NLP) models supported in the PyBuda framework, with a focus on transformer-based architectures such as Gemma-2b. It covers model architecture, component integration, optimization techniques, and practical usage examples for deploying these models on Tenstorrent hardware.

## Introduction to NLP Models in PyBuda

PyBuda supports a variety of transformer-based NLP models, enabling efficient inference on Tenstorrent hardware. The framework provides specialized optimizations for key transformer components such as attention mechanisms, embedding layers, and feed-forward networks.

The primary NLP model highlighted in this documentation is Gemma-2b, Google's lightweight language model, which is fully supported on the PyBuda framework across different hardware targets including Grayskull, Wormhole_B0, and Blackhole.

### Main Components

1.   **Rotary Position Embeddings (RoPE)**: Handles position information in the model
2.   **Attention Mechanism**: Processes relationships between tokens
3.   **RMS Normalization**: Normalizes layer outputs using root mean square
4.   **MLP (Feed-Forward Network)**: Processes token representations
