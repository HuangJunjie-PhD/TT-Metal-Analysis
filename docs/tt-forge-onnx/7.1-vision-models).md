## Purpose and Scope

This document covers the vision model support in tt-forge-onnx, detailing the 15+ vision architectures supported for image classification, object detection, and semantic segmentation tasks. For text and language models, see [Text and LLM Models](https://deepwiki.com/tenstorrent/tt-forge-onnx/7.2-text-and-llm-models). For multimodal models combining vision and language, see [Multimodal Models](https://deepwiki.com/tenstorrent/tt-forge-onnx/7.3-multimodal-models).

### ModelLoader Pattern

Each vision model follows a consistent loading pattern using the `ModelLoader` class from `third_party.tt_forge_models`:

The `ModelLoader` provides:

*   **`load_model(dtype_override=None)`**: Loads the pre-trained model with optional dtype conversion
*   **`load_inputs(dtype_override=None, input_size=None)`**: Loads and preprocesses sample inputs
*   **`post_process()` / `print_cls_results()`**: Formats and displays model outputs

## Verification Configuration

### PCC Thresholds

Different models require different Pearson Correlation Coefficient (PCC) thresholds for verification:

| Model | Typical PCC | Special Cases |
| --- | --- | --- |
| YOLOv5 | 0.99 | 0.95 for YOLOV5L at 320x320 |
| Swin | 0.99 | 0.92-0.95 for various TorchVision variants |
| ResNet | 0.99 | 0.95 for ResNet-50, ResNet-152 |
| SegFormer | 0.99 | Default verification |
| RegNet | 0.99 | 0.95 for RegNet X-8GF |
| MobileNetV2 | 0.99 | 0.97 for some HF variants |
| ViT | 0.99 | 0.93-0.98 for larger variants |
| VGG | 0.99 | 0.96-0.98 for BN variants |
| DeiT | 0.99 | 0.96 for Base variant |

The PCC threshold is configured using `VerifyConfig` and `AutomaticValueChecker`:
