## Purpose and Scope

This page documents the test suite for tensor creation operations and specialized operations in the torch_ttnn lowering pipeline. These tests verify that PyTorch operations like `torch.full`, `torch.ones`, `torch.arange`, `torch.masked_fill`, and specialized operations like `embedding` and `index` are correctly lowered to TTNN equivalents. The tests ensure both functional correctness and proper integration with the guard system and constant folding optimizations.

For element-wise operations, see [Element-wise Operation Tests](https://deepwiki.com/tenstorrent/pytorch2.0_ttnn/6.1-element-wise-and-pointwise-operation-tests). # Creation and Specialized Operation Tests

## Purpose and Scope

This page documents the test suite for tensor creation operations and specialized operations in the torch_ttnn lowering pipeline. These tests verify that PyTorch operations like `torch.full`, `torch.ones`, `torch.arange`, `torch.masked_fill`, and specialized operations like `embedding` and `index` are correctly lowered to TTNN equivalents. The tests ensure both functional correctness and proper integration with the guard system and constant folding optimizations.

For element-wise operations, see [Element-wise Operation Tests](https://deepwiki.com/tenstorrent/pytorch2.0_ttnn/6.1-element-wise-and-pointwise-operation-tests). For tensor manipulation operations like reshape and slice, see [Tensor Manipulation Tests](https://deepwiki.com/tenstorrent/pytorch2.0_ttnn/6.2-tensor-manipulation-operation-tests).

## Test Organization

The creation and specialized operation tests are organized primarily in the `tests/lowering/creation/` directory, with additional specialized patterns in `tests/pattern/` and tensor manipulation tests in `tests/lowering/tensor_manipulation/`. These tests cover operations that create new tensors, perform dtype conversions, and implement specialized operations like embedding and indexing found in production models.

### torch.masked_fill

The `torch.masked_fill` operation fills elements of a tensor with a specified value where a boolean mask is True. This operation is decomposed into a combination of arithmetic operations during lowering.

**Test Module Pattern:**

[tests/lowering/creation/test_masked_fill.py 37-42](https://github.com/tenstorrent/pytorch2.0_ttnn/blob/7ba2867e/tests/lowering/creation/test_masked_fill.py#L37-L42) defines the `MaskedFillModule` class:

**Lowering Strategy:**

The masked_fill operation is decomposed into the following TTNN operations:

1.   `ttnn.subtract` - Used to create the inverse mask
2.   `ttnn.multiply` (x2) - One for masking the input, one for the fill value
3.   `ttnn.add` - Combines the masked input with the masked fill value

The test at [tests/lowering/creation/test_masked_fill.py 10-34](https://github.com/tenstorrent/pytorch2.0_ttnn/blob/7ba2867e/tests/lowering/creation/test_masked_fill.py#L10-L34) verifies this decomposition:

1.   **Guard Check:**`can_lowering_to_ttnn(node)` must return `True` - the operation is not in a guard blocklist
