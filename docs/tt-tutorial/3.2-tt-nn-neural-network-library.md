# TT-NN Neural Network Library

Relevant source files

## Purpose and Scope

This document covers the TT-NN neural network operations library tutorial materials and documentation. TT-NN is a Python and C++ neural network OP library that provides a high-level interface for implementing neural network operations on Tenstorrent hardware.

The material covered here focuses on the tutorial content, tech reports, and educational resources for learning TT-NN. For PyTorch integration using TT-Forge and TT-Torch, see [TT-Forge and TT-Torch (PyTorch Integration)](https://deepwiki.com/tenstorrent/tt-tutorial/3.1-tt-forge-and-tt-torch-(pytorch-integration)). For low-level kernel programming, see [TT-Metalium Programming Model](https://deepwiki.com/tenstorrent/tt-tutorial/4.1-tt-metalium-programming-model).

## TT-NN Library Overview

TT-NN serves as the neural network operations layer in the Tenstorrent software stack, providing optimized implementations of common neural network primitives. The tutorial materials are organized into three main categories: foundational documentation, technical deep-dives, and hands-on tutorials.

## Integration with Tenstorrent Software Stack

### TT-NN Position in Software Hierarchy

The TT-NN library serves as the primary interface between high-level model development and low-level hardware programming. It provides:

*   **Abstraction Layer**: Simplified neural network operations that hide hardware complexity
*   **Performance Optimization**: Hardware-optimized implementations of common NN primitives
*   **Development Tools**: Profiling, tracing, and graph visualization capabilities
*   **Educational Resources**: Comprehensive tutorials and technical reports

Sources: [tt-nn/README.md 1-35](https://github.com/tenstorrent/tt-tutorial/blob/389c542c/tt-nn/README.md?plain=1#L1-L35)

## Getting Started with TT-NN

To begin learning TT-NN, users should follow the tutorial progression starting with basic tensor operations in `001.ipynb` and progressing through matrix multiplication, attention mechanisms, and advanced development tools. The technical reports provide deeper insights into specific model architectures and optimization strategies.

The tutorial materials assume familiarity with neural network concepts and basic Python programming. For users new to Tenstorrent hardware programming, it's recommended to first review the TT-Metalium fundamentals covered in [TT-Metalium Programming Model](https://deepwiki.com/tenstorrent/tt-tutorial/4.1-tt-metalium-programming-model).
