# TT-NN Visualizer

Relevant source files
*   [tt-nn visualizer/README.md](https://github.com/tenstorrent/tt-tutorial/blob/389c542c/tt-nn%20visualizer/README.md?plain=1)

This document covers the TT-NN Visualizer, a comprehensive analysis tool for visualizing and debugging model execution on Tenstorrent hardware. The TT-NN Visualizer provides interactive visualization capabilities for understanding model performance, memory usage, and execution flow within the TT-NN framework.

For information about general model visualization and graph exploration, see [Model Explorer](https://deepwiki.com/tenstorrent/tt-tutorial/6.1-model-explorer). For broader TT-NN library usage and operations, see [TT-NN Neural Network Library](https://deepwiki.com/tenstorrent/tt-tutorial/3.2-tt-nn-neural-network-library).

## Purpose and Scope

The TT-NN Visualizer serves as a debugging and performance analysis tool specifically designed for models running through the TT-NN neural network library. It provides developers with detailed insights into model execution patterns, memory utilization, tensor operations, and buffer management across Tenstorrent hardware platforms.

## Core Analysis Features

The TT-NN Visualizer provides several key visualization and analysis capabilities for model execution debugging:

| Feature | Purpose | Description |
| --- | --- | --- |
| **Interactive Graphs** | Execution Flow | Visual representation of model execution paths and operation dependencies |
| **Memory Plots** | Resource Analysis | Detailed visualization of memory allocation, usage patterns, and buffer states |
| **Tensor Details** | Data Inspection | In-depth analysis of tensor shapes, types, and data flow through operations |
| **Buffer Overviews** | Memory Management | Comprehensive view of buffer allocation, lifecycle, and optimization opportunities |
| **Operation Flow Graphs** | Performance Analysis | Visual mapping of operation sequences and execution timing |

**Sources:**[tt-nn visualizer/README.md 12](https://github.com/tenstorrent/tt-tutorial/blob/389c542c/tt-nn%20visualizer/README.md?plain=1#L12-L12)

## Report Loading and Multi-Instance Support

The TT-NN Visualizer supports flexible report loading mechanisms to accommodate different development and deployment scenarios:

The visualizer supports:

*   **File-based Loading**: Direct access to locally stored execution reports and trace files
*   **SSH-based Loading**: Remote querying capabilities for accessing reports on remote Tenstorrent systems
*   **Multi-Instance Support**: Concurrent analysis of multiple model execution instances for comparative analysis

## Integration with TT-NN Development Workflow

The TT-NN Visualizer operates as a post-execution analysis tool within the broader TT-NN development ecosystem. It processes execution artifacts generated during model runs to provide insights into:

*   **Operation Performance**: Timing analysis and bottleneck identification across TT-NN operations
*   **Memory Efficiency**: Buffer utilization patterns and memory optimization opportunities
*   **Execution Correctness**: Tensor validation and data flow verification
*   **Multi-Device Coordination**: Analysis of operations across multiple Tenstorrent devices

The tool serves as a critical component for developers optimizing model performance and debugging execution issues within the TT-NN framework.
