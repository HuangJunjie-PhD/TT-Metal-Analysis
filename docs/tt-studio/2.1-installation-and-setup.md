This document provides detailed instructions for installing and configuring TT-Studio using the `run.py` setup script, which is the recommended installation method. TT-Studio is a web-based platform for deploying and managing AI models with optimizations for Tenstorrent hardware.

## Prerequisites

Before installing TT-Studio, ensure you have the following prerequisites:

1.   **Python 3.8+**: Required to run the setup script. [README.md 28](https://github.com/tenstorrent/tt-studio/blob/a6d347af/README.md?plain=1#L28-L28)
2.   **Docker & Docker Compose**: TT-Studio uses Docker for containerization. You must have Docker and the Compose plugin installed. [README.md 29](https://github.com/tenstorrent/tt-studio/blob/a6d347af/README.md?plain=1#L29-L29)[startup.sh 82-90](https://github.com/tenstorrent/tt-studio/blob/a6d347af/startup.sh#L82-L90)
3.   **Docker Group Setup**: Your user must be in the `docker` group to run commands without `sudo`. [README.md 31-46](https://github.com/tenstorrent/tt-studio/blob/a6d347af/README.md?plain=1#L31-L46)
4.   **Tenstorrent Software**: Drivers and system configuration must be completed via the Tenstorrent Getting Started Guide. [README.md 20-24](https://github.com/tenstorrent/tt-studio/blob/a6d347af/README.md?plain=1#L20-L24)

## Installation Modes

TT-Studio supports three distinct setup modes to balance ease of use and security.

### 1. Easy Mode (`--easy`)

Designed for first-time users and quick evaluation. It minimizes prompts by using pre-configured insecure defaults for everything except the Hugging Face token.

*   **Prompts**: Only `HF_TOKEN`. [docs/run-py-guide.md 99-100](https://github.com/tenstorrent/tt-studio/blob/a6d347af/docs/run-py-guide.md?plain=1#L99-L100)
*   **Defaults**: Uses `test-secret-456` for JWT and `django-insecure-default` for Django. [docs/run-py-guide.md 110-111](https://github.com/tenstorrent/tt-studio/blob/a6d347af/docs/run-py-guide.md?plain=1#L110-L111)
*   **Artifacts**: Saves a record in `.tt_studio_easy_config.json`. [run.py 98](https://github.com/tenstorrent/tt-studio/blob/a6d347af/run.py#L98-L98)[docs/run-py-guide.md 125-133](https://github.com/tenstorrent/tt-studio/blob/a6d347af/docs/run-py-guide.md?plain=1#L125-L133)

### 2. Standard Mode (Default)

Recommended for production-like local deployments.

*   **Prompts**: Requires manual entry of `JWT_SECRET`, `HF_TOKEN`, `DJANGO_SECRET_KEY`, and `TAVILY_API_KEY`. [docs/run-py-guide.md 24-29](https://github.com/tenstorrent/tt-studio/blob/a6d347af/docs/run-py-guide.md?plain=1#L24-L29)
*   **Security**: No default secrets are provided.

### 3. Development Mode (`--dev`)

Optimized for contributors with live-reloading enabled.

*   **Hot Reload**: Mounts `./backend` and `./frontend` volumes directly into containers. [app/README.md 59-60](https://github.com/tenstorrent/tt-studio/blob/a6d347af/app/README.md?plain=1#L59-L60)
*   **Defaults**: Suggests development defaults for rapid setup. [run.py 18](https://github.com/tenstorrent/tt-studio/blob/a6d347af/run.py#L18-L18)

Sources: [docs/run-py-guide.md 64-152](https://github.com/tenstorrent/tt-studio/blob/a6d347af/docs/run-py-guide.md?plain=1#L64-L152)[run.py 17-26](https://github.com/tenstorrent/tt-studio/blob/a6d347af/run.py#L17-L26)

## Cleanup Procedures

To stop services and manage resources, `run.py` provides specialized cleanup flags.

*   **Standard Cleanup**: `python3 run.py --cleanup`
    *   Stops and removes all Docker containers and networks. [README.md 81-83](https://github.com/tenstorrent/tt-studio/blob/a6d347af/README.md?plain=1#L81-L83)
    *   Preserves data in the persistent volume. [README.md 85](https://github.com/tenstorrent/tt-studio/blob/a6d347af/README.md?plain=1#L85-L85)

*   **Full Cleanup**: `python3 run.py --cleanup-all`
    *   Removes containers, networks, and **all persistent data** (including model weights and `.env`). [run.py 20](https://github.com/tenstorrent/tt-studio/blob/a6d347af/run.py#L20-L20)

Sources: [run.py 19-20](https://github.com/tenstorrent/tt-studio/blob/a6d347af/run.py#L19-L20)[README.md 79-85](https://github.com/tenstorrent/tt-studio/blob/a6d347af/README.md?plain=1#L79-L85)[app/README.md 27-36](https://github.com/tenstorrent/tt-studio/blob/a6d347af/app/README.md?plain=1#L27-L36)
