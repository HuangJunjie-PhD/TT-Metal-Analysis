## Purpose and Scope

This document provides an overview of Tenstorrent's Add-In Boards (AIBs), which are the core hardware components that deliver Tenstorrent's AI acceleration capabilities. These PCIe-based cards contain Tensix processors and form the building blocks of Tenstorrent's hardware ecosystem. For information about complete systems that incorporate these AIBs, see [Complete Systems](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.1-complete-systems).

## AIB Product Portfolio

Tenstorrent offers several generations of AIBs with different performance characteristics, along with accessories that enhance their functionality and integration capabilities.

### Processor Generations

Tenstorrent offers three main generations of Tensix processors:

1.   **Blackhole Series** - The latest generation featuring PCIe 5.0, 120 Tensix cores, and SiFive x280 "Big RISC-V" cores [core/aibs/blackhole/specifications.md 13-19](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/specifications.md?plain=1#L13-L19)
    *   Models: p100a (Active), p150a (Active + Networking), p150b (Passive + Networking).
    *   For detailed information, see [Blackhole Processors](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.2.1-blackhole-processors).

2.   **Wormhole Series** - Mid-generation processors featuring PCIe 4.0 and 200G QSFP-DD networking [core/aibs/wormhole/specifications.md 30-32](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L30-L32)
    *   Models: n150d, n150s, n300d, n300s.
    *   For detailed information, see [Wormhole Processors](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.2.2-wormhole-processors).

3.   **Grayskull Series** - The first generation, now deprecated. Requires legacy software versions [core/aibs/grayskull/specifications.md 15-19](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/grayskull/specifications.md?plain=1#L15-L19)
    *   Models: e75, e150.
    *   For detailed information, see [Grayskull Processors](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.2.3-grayskull-processors).

### Accessories

Tenstorrent provides accessories to enhance the functionality of their AIBs:

1.   **Warp 100 Bridge** - Internal bridge for inter-card connectivity on Wormhole cards [core/aibs/wormhole/specifications.md 49](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/specifications.md?plain=1#L49-L49)
2.   **Active Cooling Kit (TX-01001)** - Duct and blower fan assembly for passive cards (e150, n150s, n300s) used in desktop environments [core/aibs/wormhole/ack.md 1-13](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/ack.md?plain=1#L1-L13)

For detailed information, see [Accessories](https://deepwiki.com/tenstorrent/tenstorrent.github.io/2.2.4-accessories).

## Data Precision Format Support

Tenstorrent AIBs support a variety of data formats for AI computation. Blackhole adds support for higher precision on its "Big RISC-V" cores.

| Format | Grayskull™ | Wormhole™ | Blackhole™ (Tensix) |
| --- | --- | --- | --- |
| FP8, FP16, BFLOAT16 | Yes | Yes | Yes |
| FP32 | No | Output Only | Output Only |
| BFP2, BFP4, BFP8 | Yes | Yes | Yes |
| INT8 | No | Yes | Yes |
| INT32 | No | Output Only | Output Only |
| TensorFloat (TF32) | No | Yes | Yes |
| Vector (VTF19, VFP32) | VTF19 Only | Yes | Yes |

## System Architecture and Integration

### AIB System Integration

Tenstorrent AIBs integrate into host systems via PCIe. Blackhole cards specifically require ATX 3.1 certified power supplies and 12V-2x6 connectors [core/aibs/blackhole/specifications.md 39-40](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/specifications.md?plain=1#L39-L40)

### Deployment Configurations

Tenstorrent AIBs can be deployed in various configurations depending on performance requirements:

*   **Active Cooled (p100a, p150a, n150d, n300d):** Intended for conventional desktop systems [core/aibs/blackhole/support.md 14-15](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L14-L15)
*   **Passive Cooled (p150b, n150s, n300s):** Intended for rack-mounted servers with high static pressure airflow [core/aibs/blackhole/support.md 16](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L16-L16)

## Installation Overview

1.   **Hardware Installation**
    *   Verify PCIe slot compatibility (Gen 4.0 for Wormhole/Grayskull, Gen 5.0 for Blackhole).
    *   Ensure proper power delivery (EPS12V for Wormhole, 12V-2x6 for Blackhole).
    *   Install Active Cooling Kit if using passive cards in a desktop [core/aibs/wormhole/ack.md 11-13](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/ack.md?plain=1#L11-L13)

2.   **Software Setup**
    *   Install `tt-kmd` driver.
    *   Update firmware using `tt-flash` or `fw_pack`.
    *   Verify enumeration with `tt-smi`.
