## High-Level Architecture Overview

The tt-perf-report tool is a command-line utility designed to analyze performance traces from TT-Metal operations. Its architecture consists of several interconnected components that process CSV trace data, analyze different operation types, classify bottlenecks, and generate optimization advice.

## Core Algorithms and Formulas

### Performance Metric Calculations

The system calculates several key performance metrics:

1.   **DRAM Bandwidth**:

```
DRAM Speed (GB/s) = Total data size (bytes) / Duration (s) / 1e9
```
2.   **DRAM Utilization**:

```
DRAM Percentage = (DRAM Speed (GB/s) / 288) * 100
```
3.   **FLOPs**:

```
FLOPs = (M * K * N * W * Z * 2) / Duration (s)
```
4.   **FLOPs Utilization**:

```
FLOPs Percentage = (FLOPs / Peak FLOPs) * 100
```

where `Peak FLOPs = tflops_per_core(math_fidelity) * 1e12 * core_count`

Sources: [src/tt_perf_report/perf_report.py 208-277](https://github.com/tenstorrent/tt-perf-report/blob/076a1b39/src/tt_perf_report/perf_report.py#L208-L277)[src/tt_perf_report/perf_report.py 51-60](https://github.com/tenstorrent/tt-perf-report/blob/076a1b39/src/tt_perf_report/perf_report.py#L51-L60)
