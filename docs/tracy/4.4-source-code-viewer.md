The Source Code Viewer is Tracy's integrated source code and assembly viewer that provides a unified interface for examining application source code alongside performance profiling data. It supports syntax-highlighted source display, real-time assembly disassembly, and overlays performance metrics directly onto code lines to enable precise performance analysis at the instruction level.

For information about the broader profiler application architecture, see [Application Overview](https://deepwiki.com/tenstorrent/tracy/4.1-application-overview). For details on performance data processing, see [Data Processing Engine](https://deepwiki.com/tenstorrent/tracy/4.2-data-processing-engine).

## Core Components

### SourceView Class

The `SourceView` class serves as the main controller for the source code viewer, managing all display modes and coordinating between different subsystems.

| Component | Purpose | Key Members |
| --- | --- | --- |
| Display Mode Management | Controls source/assembly/mixed views | `m_displayMode`, `SelectViewMode()` |
| File Operations | Source file loading and parsing | `ParseSource()`, `OpenSource()`, `OpenSymbol()` |
| Assembly Integration | Disassembly and instruction analysis | `Disassemble()`, `m_asm`, `m_cpuArch` |
| Performance Overlay | Integrates profiling data with code | `GatherIpStats()`, `GatherIpHwStats()` |
| Navigation | History and line selection | `m_history`, `SelectLine()`, `SelectAsmLines()` |

| --- | --- | --- |

| Level Assignment | Jump range sorting and level calculation | Prevent overlapping jump visualizations |

The viewer maintains navigation history and provides multiple ways to navigate through code.

| Navigation Method | Implementation | Purpose |
| --- | --- | --- |
| Line Selection | `SelectLine()`, `m_selectedLine` | Direct line navigation |
| Address Selection | `SelectAsmLines()`, `m_selectedAddresses` | Assembly address targeting |
| History Management | `m_history`, `m_historyCursor` | Back/forward navigation |
| Symbol Opening | `OpenSymbol()` with history updates | Function-level navigation |
