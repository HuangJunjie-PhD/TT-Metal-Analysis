This document provides technical specifications, requirements, and setup information for the Tenstorrent T1000 Desktop Workstation. The T1000 is an entry-level AI acceleration platform featuring a single Wormhole n150s Tensix processor, designed for developers and researchers working with Tenstorrent's AI hardware.

For information about multi-processor systems, see [TT-LoudBox (T3000)](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/TT-LoudBox%20(T3000))[T7000](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/T7000) or [TT-QuietBox](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/TT-QuietBox)

## System Overview

The T1000 (model TW-01001) is Tenstorrent's entry-level workstation built on the SuperMicro SuperServer SYS-E403-12P-FN2T platform [core/systems/t1000/specifications.md 21](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t1000/specifications.md?plain=1#L21-L21) It contains a single Wormhole n150s Tensix processor [core/systems/t1000/specifications.md 17](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t1000/specifications.md?plain=1#L17-L17) and serves as the most compact system in Tenstorrent's hardware portfolio.

Sources: [core/systems/t1000/README.md 1-3](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t1000/README.md?plain=1#L1-L3)[core/systems/t1000/specifications.md 11-22](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t1000/specifications.md?plain=1#L11-L22)

## Software Stack Integration

The T1000 hardware is managed and programmed via the Tenstorrent software stack. The `tt-kmd` (Kernel Mode Driver) manages the `n150s` device over the PCIe interface, while `tt-metalium` provides the low-level programming interface.

Sources: Software setup requirements from [core/systems/t1000/specifications.md 38-40](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t1000/specifications.md?plain=1#L38-L40)[core/systems/t1000/support.md 13-15](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/systems/t1000/support.md?plain=1#L13-L15)

## Operating System Requirements
