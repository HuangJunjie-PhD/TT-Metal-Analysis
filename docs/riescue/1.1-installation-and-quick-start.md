### Required External Dependencies

RiESCUE requires the following non-Python dependencies to be installed and available in your `PATH`:

| Dependency | Purpose | Repository |
| --- | --- | --- |
| **RISC-V GNU Toolchain** | Compiles assembly tests to ELF executables | [riscv-gnu-toolchain](https://github.com/tenstorrent/riescue/blob/07cefde3/riscv-gnu-toolchain) |
| **Whisper ISS** | Tenstorrent's instruction set simulator | [whisper](https://github.com/tenstorrent/riescue/blob/07cefde3/whisper) |
| **Spike ISS** | Reference RISC-V ISS | [riscv-isa-sim](https://github.com/tenstorrent/riescue/blob/07cefde3/riscv-isa-sim) |

The toolchain must provide these executables:

* `riscv64-unknown-elf-gcc` - Compiler (referenced in [riescue/lib/toolchain.py](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/lib/toolchain.py#LNaN-LNaN))
* `riscv64-unknown-elf-objdump` - Disassembler (referenced in [riescue/lib/toolchain.py](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/lib/toolchain.py#LNaN-LNaN))
* `whisper` - Whisper simulator (referenced in [riescue/lib/toolchain.py](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/lib/toolchain.py#LNaN-LNaN))
* `spike` - Spike simulator (referenced in [riescue/lib/toolchain.py](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/lib/toolchain.py#LNaN-LNaN))

**Environment Configuration:**

Set `RV_TOOLCHAIN` environment variable to point to your toolchain installation:

The container system automatically exports this variable: [infra/singularity/container.py75-82](https://github.com/tenstorrent/riescue/blob/07cefde3/infra/singularity/container.py#L75-L82)

**Sources:** [README.md87-92](https://github.com/tenstorrent/riescue/blob/07cefde3/README.md?plain=1#L87-L92) [infra/singularity/container.py75-82](https://github.com/tenstorrent/riescue/blob/07cefde3/infra/singularity/container.py#L75-L82)

## Quick Start: Running Your First Test

### RiescueD - Directed Test Execution

The `riescued` command-line tool runs directed assembly tests.

**Basic Usage:**

**Command-Line Arguments:**

The `RiescueD.add_arguments()` method defines the CLI interface [riescue/riescued.py89-138](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/riescued.py#L89-L138):

| Argument | Purpose | Default |
| --- | --- | --- |
| `--testfile` / `-t` | Path to test assembly file | Required |
| `--run_dir` / `-rd` | Output directory | `./` |
| `--seed` | Random seed | Random |
| `--cpuconfig` | CPU configuration JSON | `dtest_framework/lib/config.json` |
| `--run_iss` | Execute on ISS after compilation | False |
| `--elaborate_only` | Skip compilation and simulation | False |

**Execution Flow:**

The `RiescueD.run()` method orchestrates the workflow [riescue/riescued.py188-220](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/riescued.py#L188-L220):

1. **Configure:** `RiescueD.configure()` builds `FeatMgr` from test header and CPU config [riescue/riescued.py222-237](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/riescued.py#L222-L237)
2. **Generate:** `RiescueD.generate()` produces assembly and linker script [riescue/riescued.py239-267](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/riescued.py#L239-L267)
3. **Build:** `RiescueD.build()` compiles to ELF and disassembles [riescue/riescued.py269-309](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/riescued.py#L269-L309)
4. **Simulate:** `RiescueD.simulate()` runs on Whisper or Spike (if `--run_iss`) [riescue/riescued.py311-411](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/riescued.py#L311-L411)

**Sources:** [riescue/riescued.py31-220](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/riescued.py#L31-L220) [README.md24-34](https://github.com/tenstorrent/riescue/blob/07cefde3/README.md?plain=1#L24-L34)

## Generated Output Files

After running `riescued`, the following files are created by the `GeneratedFiles` class:

File paths are managed by `GeneratedFiles.from_testname()` [riescue/dtest\_framework/artifacts.py](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/dtest_framework/artifacts.py):

| File | Attribute | Purpose |
| --- | --- | --- |
| `test.S` | `generated_files.assembly` | Elaborated assembly with directives processed |
| `test.ld` | `generated_files.linker_script` | Linker script with memory layout |
| `test.elf` | `generated_files.elf` | Compiled executable |
| `test.dis` | `generated_files.dis` | Human-readable disassembly |

**Sources:** [riescue/dtest\_framework/artifacts.py](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/dtest_framework/artifacts.py) [riescue/riescued.py61](https://github.com/tenstorrent/riescue/blob/07cefde3/riescue/riescued.py#L61-L61)
