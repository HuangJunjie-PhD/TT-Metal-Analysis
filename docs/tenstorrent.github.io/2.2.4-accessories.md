This page documents the accessories available for Tenstorrent Tensix processors, focusing on the Warp 100 Bridge for high-speed inter-processor communication, the Active Cooling Kit (TX-01001) for thermal management, and validated networking cables for the Blackhole™ architecture.

## Overview

Tenstorrent offers several primary accessories to enhance the functionality, thermal performance, and connectivity of Tensix processor cards:

1.   **Warp 100 Bridge (TX-01002)**: Enables high-speed direct communication between multiple Wormhole processors.
2.   **Active Cooling Kit (TX-01001)**: Provides supplementary cooling for passive-cooled Wormhole and Grayskull cards in desktop environments.
3.   **QSFP-DD 800G Cables**: Validated passive copper cables for networking Blackhole™ p150a and p150b cards.

## Warp 100 Bridge

The Warp 100 Bridge (TX-01002) is a hardware component that creates high-speed direct connections between multiple Wormhole Tensix processors, enabling enhanced parallel processing capabilities without traversing the PCIe bus.

The Warp 100 Bridge is compatible with the following Wormhole Tensix processor cards:

| Wormhole™ n300d | TC-02004 | Active | Up to 2 per card* |
| Wormhole™ n300s | TC-02003 | Passive | Up to 2 per card* |

_*n300d and n300s cards feature 2.5-slot coolers and require the triple-slot Warp 100 Bridge variant._

## Active Cooling Kit (TX-01001)

The Active Cooling Kit provides a mechanical duct and blower fan assembly to convert passive-cooled cards for use in desktop workstations where high static pressure airflow is unavailable.

### Compatibility

The Active Cooling Kit is compatible with the following Tensix processor cards:

| Card Model | Part Number | Base Cooler Type |
| --- | --- | --- |
| Grayskull™ e150 | TC-01002 | Passive |
| Wormhole™ n150s | TC-02001 | Passive |
| Wormhole™ n300s | TC-02003 | Passive |

Sources: [core/aibs/wormhole/ack.md 3-10](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/wormhole/ack.md?plain=1#L3-L10)

## Blackhole™ Networking Accessories

Unlike the proprietary Warp 100 Bridge used by Wormhole, Blackhole™ p150a and p150b cards utilize industry-standard QSFP-DD ports for inter-card networking.

### Validated 800G QSFP-DD Cables

Tenstorrent has validated the following passive copper cables for 800 Gbps connectivity between Blackhole™ cards:

| Manufacturer | Model | Length |
| --- | --- | --- |
| Amphenol | SF-NJYYER0006-000.5M | 0.5m (1.6 ft.) |
| Amphenol | SF-NJYYEK0001-001M | 1m (3.3 ft.) |
| FS | QDD-800G-PC005 | 0.5m (2 ft.) |
| FS | QDD-800G-PC01 | 1m (3 ft.) |

Sources: [core/aibs/blackhole/support.md 18-25](https://github.com/tenstorrent/tenstorrent.github.io/blob/eda54884/core/aibs/blackhole/support.md?plain=1#L18-L25)
