The visualization and reporting system provides interactive HTML reports and data visualization capabilities for performance analysis results. This system transforms simulation outputs and benchmark comparisons into user-friendly interactive dashboards with filtering, sorting, and export capabilities.

For information about the underlying performance analysis that generates the data visualized here, see [Performance Metrics Extraction](https://deepwiki.com/tenstorrent/polaris/3.1-performance-metrics-extraction) and [Correlation Analysis](https://deepwiki.com/tenstorrent/polaris/3.2-correlation-analysis). For details on projection comparison data processing, see [Projection Comparison](https://deepwiki.com/tenstorrent/polaris/3.3-projection-comparison).

## HTML Report Generation System

The system generates two primary types of HTML reports using template-based rendering:

### Job Summary Reports

Job summary reports provide high-level comparison statistics between projection runs using the `template-jobsummary.html` template. These reports display job-level match statistics, exact matches, approximate matches, and mismatches in a tabular format.

Sources: [tools/templates/template-jobsummary.html 1-67](https://github.com/tenstorrent/polaris/blob/ac6ed4a3/tools/templates/template-jobsummary.html#L1-L67)

### Performance Comparison Reports

Performance comparison reports provide detailed layer-wise analysis using the `template-cmpperf-summary.html` template. These reports integrate Google Charts for interactive visualization and filtering.

Sources: [tools/templates/template-cmpperf-summary.html 1-197](https://github.com/tenstorrent/polaris/blob/ac6ed4a3/tools/templates/template-cmpperf-summary.html#L1-L197)

## Template System and Data Integration

### Template Variable Processing

The template system uses variable substitution to inject dynamic content:

| Template | Key Variables | Purpose |
| --- | --- | --- |
| `template-jobsummary.html` | `job_columns`, `job_dataset`, `run1_name`, `run2_name` | Job comparison statistics |
| `template-cmpperf-summary.html` | `summary_table`, `ratio_name`, `diff_name`, `dir1name`, `dir2name` | Performance comparison details |
