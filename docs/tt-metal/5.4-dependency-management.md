## Purpose and Scope

This document describes the integration of third-party dependencies in the `tt-metal` build system. It documents system packages, C++ libraries managed via CPM (CMake Package Manager), the SFPI firmware toolchain, and UMD integration. The dependency management system ensures consistency across different hardware platforms (Wormhole, Blackhole) and build configurations (Release, Debug, ASan, TSan).

## Dependency Architecture Overview

The repository manages dependencies across multiple layers, from system-level libraries to high-level Python packages. The top-level `CMakeLists.txt` orchestrates these by including specialized scripts for third-party management.

### UMD (Unified Metalium Driver)

The UMD is the foundational hardware driver, integrated via submodules and direct header inclusion.

*   **Integration**: UMD provides low-level hardware access, chip discovery, and PCIe/Ethernet communication primitives.
*   **Cluster Discovery**: `tt_cluster.cpp` heavily utilizes UMD classes like `tt::umd::Cluster`, `tt::umd::ClusterDescriptor`, and `tt::umd::SocDescriptor` to discover hardware topology [tt_metal/llrt/tt_cluster.cpp 38-45](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/tt_cluster.cpp#L38-L45)
*   **Hardware Support**: UMD provides the architecture types (`tt::ARCH`) and core coordinate systems used throughout the stack [tt_metal/llrt/tt_cluster.cpp 42](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/tt_cluster.cpp#L42-L42)[tt_metal/llrt/tt_cluster.hpp 26-36](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/tt_cluster.hpp#L26-L36)

**Sources:**[tt_metal/llrt/tt_cluster.cpp 38-45](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/tt_cluster.cpp#L38-L45)[tt_metal/llrt/tt_cluster.hpp 26-36](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/tt_cluster.hpp#L26-L36)

### SFPI Toolchain

The SFPI (Software-Firmware Programming Interface) toolchain is crucial for compiling device-side kernels.

*   **Discovery**: The `JitBuildEnv` in `jit_build/build.cpp` searches for SFPI at `runtime/sfpi` or `/opt/tenstorrent/sfpi`[tt_metal/jit_build/build.cpp 125-133](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/jit_build/build.cpp#L125-L133)
*   **Compiler**: Uses `riscv-tt-elf-g++` for compiling RISC-V firmware and kernels [tt_metal/jit_build/build.cpp 129-131](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/jit_build/build.cpp#L129-L131)
*   **CCache Integration**: Kernel builds can use `ccache` if `TT_METAL_CCACHE_KERNEL_SUPPORT` is enabled [tt_metal/jit_build/build.cpp 115-120](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/jit_build/build.cpp#L115-L120)

**Sources:**[tt_metal/jit_build/build.cpp 115-140](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/jit_build/build.cpp#L115-L140)

## Runtime Configuration Options

Runtime behavior and dependency utilization are controlled via `RunTimeOptions` and environment variables defined in `rtoptions.cpp`.

*   **Cache Paths**: `TT_METAL_CACHE` defines where compiled kernels are stored [tt_metal/llrt/rtoptions.cpp 52](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/rtoptions.cpp#L52-L52)
*   **Feature Toggles**: Variables like `TT_METAL_DEVICE_PROFILER` or `TT_METAL_WATCHER_ENABLED` control the activation of optional dependencies like Tracy or the Watcher system [tt_metal/llrt/rtoptions.cpp 123](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/rtoptions.cpp#L123-L123)[tt_metal/llrt/rtoptions.hpp 98](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/rtoptions.hpp#L98-L98)
*   **Hardware Simulation**: `TT_METAL_SIMULATOR` points to the simulator library/executable [tt_metal/llrt/rtoptions.cpp 55](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/rtoptions.cpp#L55-L55)
*   **Fabric and MGD**: `TT_MESH_GRAPH_DESC_PATH` and `TT_METAL_USE_MGD_2_0` control fabric topology dependencies [tt_metal/llrt/rtoptions.cpp 60](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/rtoptions.cpp#L60-L60)[tt_metal/llrt/rtoptions.cpp 113](https://github.com/tenstorrent/tt-metal/blob/f30f8df0/tt_metal/llrt/rtoptions.cpp#L113-L113)
